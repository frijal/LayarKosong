Kebijakan Keamanan

Halaman ini menjelaskan bagaimana Layar Kosong menangani isu keamanan, pelaporan kerentanan, serta permintaan penghapusan data atau konten.

Pelaporan Kerentanan

Jika kamu menemukan potensi celah keamanan atau kesalahan konfigurasi, silakan laporkan dengan itikad baik melalui:

  • Email: sini@dalam.web.id
  • Formulir: Formulir Pelaporan

Penghapusan Data

Permintaan penghapusan data atau koreksi konten dapat diajukan melalui kebijakan dan formulir berikut:

  • Kebijakan Penghapusan Data
  • Formulir Permintaan

Security Policy

This page explains how Layar Kosong handles security issues, vulnerability reporting, and requests related to data or content removal.

Vulnerability Reporting

If you discover a potential security issue or misconfiguration, please report it responsibly via:

  • Email: sini@dalam.web.id
  • Form: Reporting Form

Data Removal

Requests for data removal or content correction can be submitted through:

  • Data Deletion Policy
  • Removal Request Form
