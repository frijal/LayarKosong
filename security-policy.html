Security Policy

Security & Vulnerability Reporting

This document complements the technical guidelines found in the /.well-known/security.txt file.

Layar Kosong is committed to maintaining data security and visitor comfort. We appreciate the efforts of security researchers who find vulnerabilities in our system and report them responsibly.

Vulnerability Reporting

If you discover a potential security issue or misconfiguration, please report it in good faith via:

  • Email: sini@dalam.web.id
  • Form: Security Reporting Form

We highly appreciate reports that include sufficient technical details to reproduce the findings.
