Refresh GitHub Codespaces dari Browser

Saat lingkungan pengembangan Anda di Codespaces terasa aneh atau error, ada beberapa cara untuk menyegarkannya kembali. Panduan ini menjelaskan perbedaan antara Restart, Rebuild, dan Full Rebuild, serta kapan harus menggunakannya.

Cara Rebuild Codespace dari Browser

  1. Buka Codespace Anda di browser (VS Code web).
  2. Buka Command Palette dengan menekan Ctrl+Shift+P (Windows/Linux) atau Cmd+Shift+P (Mac).
  3. Ketik "Rebuild" pada Command Palette.
  4. Pilih salah satu dari dua opsi: **Rebuild Container** (lebih cepat, menggunakan cache) atau **Full Rebuild Container** (bersih total, lebih lama).

Perbedaan Antar Metode Refresh

ModeApa yang DilakukanKapan Digunakan
**Restart**Memulai ulang kontainer yang sudah ada. Cepat.Untuk masalah minor atau aplikasi yang hang.
**Rebuild**Membangun ulang kontainer, tetapi menggunakan cache dari image sebelumnya.Saat ada perubahan kecil pada file konfigurasi `devcontainer.json`.
**Full Rebuild**Menghapus semua cache dan membangun ulang kontainer dari nol. Paling lama.Saat terjadi error aneh, dependensi rusak, atau ada perubahan besar pada konfigurasi.
