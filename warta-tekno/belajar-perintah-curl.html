Cheat Sheet Perintah cURL Lengkap

cURL adalah tool baris perintah yang super canggih untuk transfer data. Alat ini sangat membantu developer dan tester untuk mengirim request, menguji API, dan mengunduh file. Berikut adalah kumpulan perintah cURL yang paling berguna.

Operasi Dasar HTTP GET

Mengambil konten URL

curl http://example.com

Menyimpan output ke file

curl -o output.html http://example.com

Mengirim Header Kustom

curl -H "Authorization: Bearer YOUR_TOKEN" http://api.example.com

Mengikuti Pengalihan (Redirect)

curl -L http://example.com

Operasi HTTP POST/PUT

Mengirim data form

curl -X POST -d "key1=val1&key2=val2" http://example.com/form

Mengirim data JSON

curl -X POST -d '{"key":"value"}' -H "Content-Type: application/json" http://api.example.com

Mengunggah file

curl -X POST -F "file=@/path/to/yourfile.txt" http://example.com/upload

Operasi Unduh File

Mengunduh dengan nama file asli

curl -O http://example.com/file.zip

Melanjutkan unduhan yang terputus

curl -C - -O http://example.com/largefile.zip
