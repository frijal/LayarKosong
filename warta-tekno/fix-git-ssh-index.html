Solusi untuk error: cannot run ssh: No such file or directory pada Linux, Windows, dan macOS.

Slackware & Keluarganya

Panduan menggunakan slackpkg untuk menginstal git dan openssh. Termasuk cara cek selinux & izin SSH agent.

Arch Linux & Turunannya

Gunakan pacman untuk memastikan paket openssh dan git terinstal. Termasuk script perbaikan otomatis dengan log dan dry-run.

Debian, Ubuntu, Mint

Perbaikan dengan apt install git openssh-client, pengecekan PATH, dan script otomatis yang kompatibel untuk sistem Debian-based.

Fedora, RHEL, CentOS

Panduan menggunakan dnf atau yum untuk menginstal git dan openssh. Termasuk cara cek selinux & izin SSH agent.

Windows (CMD & PowerShell)

Langkah untuk memperbaiki SSH di Git Bash dan PowerShell: pastikan OpenSSH Client aktif, perbaiki PATH, dan jalankan test koneksi.

macOS

Gunakan xcode-select --install untuk mengaktifkan Command Line Tools dan pastikan /usr/bin/ssh terdeteksi di PATH Git.
