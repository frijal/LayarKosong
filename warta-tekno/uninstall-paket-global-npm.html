Cara Gampang Uninstall Paket Global Node.js (npm, yarn, pnpm)

Sudah install banyak tool global tapi bingung cara hapusnya? Panduan santai ini bakal ngebantu kamu membersihkan paket-paket Node.js yang udah nggak kepake—di Linux, macOS, maupun Windows. Gampang, kok!

Pernah nggak sih, kamu iseng-install berbagai tool global di Node.js karena lihat tutorial atau pengen coba-coba? Lama-lama daftar paket jadi menumpuk, dan kamu lupa sebenernya paket itu masih dipake apa nggak. Atau mungkin kamu salah install dan bingung cara hapusnya. Tenang, ini adalah masalah yang wajar banget.

Untungnya, membersihkan paket global itu jauh lebih mudah daripada yang kamu bayangkan. Perintahnya mirip banget sama waktu kamu install. Cukup ganti kata install jadi uninstall atau remove, dan selesai! Di panduan ini, kita bakal bahas cara hapus paket global pakai tiga package manager paling populer: npm, yarn, dan pnpm—di semua sistem operasi.

Sebelum mulai, pastikan kamu udah tahu dulu paket mana aja yang terpasang secara global. Kamu bisa pake perintah npm list -g --depth=0 (atau versi yarn/pnpm-nya) buat lihat daftarnya. Setelah itu, tinggal pilih paket yang mau dihapus, dan ikuti langkah-langkah di bawah ini.

1. Terminal (Linux & macOS) 🐧🍎

Buat kamu pengguna Linux (Ubuntu, Fedora, BlankOn, CachyOS, atau distro apapun) dan macOS, kabar baik: perintahnya sama persis. Nggak ada perbedaan berarti. Yuk, kita bahas satu per satu.

Untuk NPM

NPM punya perintah uninstall yang bisa langsung dipakai. Tambahkan flag -g (atau --global) biar paket yang kehapus adalah yang terinstall secara global, bukan lokal.

npm uninstall -g <nama-paket>

Ganti <nama-paket> dengan nama tool yang mau kamu hapus. Misal npm uninstall -g nodemon.

Untuk Yarn

Yarn punya sintaks yang sedikit berbeda, tapi nggak kalah simpel. Pakai perintah global remove.

yarn global remove <nama-paket>

Lebih deskriptif, ya? Global remove jelas banget maksudnya.

Untuk PNPM

PNPM mengikuti gaya NPM. Kamu bisa pake remove atau rm, plus flag -g.

pnpm remove -g <nama-paket>

Catatan Penting (Linux/macOS): Kalau dulu kamu install paket-paket ini pake sudo (misal: sudo npm install -g ...), maka kamu harus pakai sudo juga waktu uninstall. Contoh: sudo npm uninstall -g <nama-paket>. Ini karena paket global sering disimpan di direktori sistem yang dilindungi.

2. Windows (PowerShell) 💻

Pengguna Windows, jangan khawatir! Perintahnya tetap sama di PowerShell. Kamu nggak perlu belajar sintaks baru.

Untuk NPM

npm uninstall -g <nama-paket>

Untuk Yarn

yarn global remove <nama-paket>

Untuk PNPM

pnpm remove -g <nama-paket>

Catatan Penting (Windows): Paket global di Windows biasanya terinstall di folder yang dilindungi sistem, seperti C:\Program Files\nodejs. Karena itu, kamu mungkin perlu menjalankan PowerShell sebagai Administrator (klik kanan ikon PowerShell > "Run as administrator") biar proses hapusnya lancar. Ini mirip kayak sudo di Linux, tapi versi Windows-nya.

3. Contoh Praktis (Studi Kasus) 🧑💻

Biar makin jelas, kita coba studi kasus. Misalnya, beberapa waktu lalu kamu menjalankan perintah npm list -g --depth=0 dan melihat ada dua paket yang udah nggak terpakai: nodemon dan pm2. Keduanya udah nggak kamu butuhin lagi karena project lama udah selesai.

Nah, untuk membersihkannya, tinggal eksekusi perintah uninstall satu per satu:

npm uninstall -g nodemon
 npm uninstall -g pm2

Atau kalau kamu pake Yarn:

yarn global remove nodemon
 yarn global remove pm2

Atau pake PNPM:

pnpm remove -g nodemon
 pnpm remove -g pm2

Selesai! Paket-paket itu sekarang udah bersih dari laptop kamu. Gampang, 'kan? Cuma butuh beberapa detik.

Kamu juga bisa menggabungkan beberapa paket dalam satu baris perintah (untuk npm dan pnpm) dengan memisahkan paket pakai spasi, misalnya npm uninstall -g nodemon pm2. Tapi buat keamanan, lebih baik satu-satu dulu.

4. Tips Tambahan dan Perhatian ⚠️

Sebelum kamu asal hapus, ada beberapa hal kecil yang perlu diperhatikan biar prosesnya mulus.

Cek Daftar Paket Global Dulu

Jangan hapus paket kalau kamu nggak yakin fungsinya apa. Beberapa paket mungkin masih digunakan oleh tool lain secara diam-diam. Cara paling aman: cek dulu daftar paket global kamu.

  • NPM: npm list -g --depth=0
  • Yarn: yarn global list
  • PNPM: pnpm list -g --depth=0

Hati-hati dengan Dependensi

Beberapa paket global bisa saling bergantung satu sama lain. Meskipun jarang, ada kemungkinan menghapus satu paket bisa mempengaruhi paket lain. Kalau ragu, cari tahu dulu fungsi paket tersebut sebelum dihapus.

Bersihkan Cache (Opsional)

Setelah uninstall, kamu juga bisa membersihkan cache package manager untuk menghemat ruang penyimpanan. Tapi ini nggak wajib.

  • NPM: npm cache clean --force
  • Yarn: yarn cache clean
  • PNPM: pnpm store prune

Kesimpulan: Bersih-bersih Itu Menyenangkan

Membersihkan paket global Node.js itu mudah dan cepat. Cukup satu perintah di terminal atau PowerShell, dan tool yang udah nggak terpakai bisa langsung lenyap. Ini bukan cuma soal menghemat ruang penyimpanan, tapi juga bikin lingkungan develop kamu lebih rapi dan mudah dikelola.

Jadi, lain kali kamu lihat daftar paket global yang penuh dan berantakan, jangan malas buat bersih-bersih. Dengan panduan ini, kamu udah punya semua yang diperlukan: perintah untuk npm, yarn, dan pnpm, di semua sistem operasi. Selamat membersihkan!
