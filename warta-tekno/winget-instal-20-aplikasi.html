Pasang 20 Aplikasi Windows Sekaligus dengan Winget

 

Winget bikin setup Windows jadi cepat, aman, dan otomatis. + skrip PowerShell untuk memasang 20 aplikasi sekaligus, lengkap dengan tips, ekspor/impor daftar, dan best practice.

 

Winget itu apa sih?

 

Winget adalah manajer paket resmi dari Microsoft untuk Windows 10 dan 11, dirilis sejak 2020. Bisa dibilang ini “toko aplikasi” yang hidup di dalam Command Prompt atau PowerShell.

 
  • Semua paket diambil dari repositori resmi yang dikurasi komunitas.
  • Aman, karena yang dipasang adalah versi asli dari publisher.
  • Karena berbasis command line, Winget gampang diintegrasikan ke skrip atau dipakai untuk instalasi massal.
    •  
 

Instalasi massal: hemat waktu & lebih aman

 

Biasanya, kalau habis install ulang Windows, kita harus unduh installer satu per satu: browser, pemutar musik, editor teks, dan seterusnya. Itu bisa makan waktu berjam-jam, plus ada risiko salah klik link palsu yang ternyata perangkat lunak berbahaya.

 

Dengan Winget, cukup jalankan satu perintah. Contoh untuk 20 aplikasi populer:

  
winget install MarkText.MarkText AIMP.AIMP Microsoft.VisualStudioCode Notepad++.Notepad++ VideoLAN.VLC 7zip.7zip Audacity.Audacity Discord.Discord Spotify.Spotify REALiX.HWiNFO Microsoft.PowerToys BlastApps.FluentSearch Git.Git Python.Python.3 Zen-Team.Zen-Browser OBSProject.OBSStudio Notion.Notion SlackTechnologies.Slack Zoom.Zoom Valve.Steam
 

Selesai. Tinggal ditinggal beristirahat sejenak, Windows akan dapat memasang semuanya tanpa klik-klik installer.

 

Tips praktis

 
  • Nama paket harus sesuai format: Publisher.App. Misalnya Slack → SlackTechnologies.Slack, Steam → Valve.Steam.
  • Kalau lupa, pakai winget search [nama] untuk cari ID paket yang benar.
  • Ada juga opsi GUI:  
    • winstall.app → pilih aplikasi, otomatis dibuatkan perintah Winget.
    • WingetUI → antarmuka grafis, jadi tidak perlu command line.

Ekspor & impor daftar aplikasi

 

Kalau sering pindah PC, Winget juga bisa ekspor daftar aplikasi ke file JSON:

  
winget export -o daftar.json
 

Lalu di mesin baru:

  
winget import -i daftar.json
 

Semua aplikasi lama langsung terpasang lagi. Selain itu, Winget mendukung file konfigurasi YAML (DSC) untuk sekaligus mengatur aplikasi + setting Windows sesuai kebutuhan.

 

Contoh skrip PowerShell siap pakai

Simpan sebagai setup.ps1, lalu jalankan dengan hak Administrator agar mulus.

# setup.ps1 # Skrip instalasi massal aplikasi Windows dengan Winget # Jalankan dengan hak administrator agar semua aplikasi bisa terpasang Write-Host "=== Instalasi aplikasi favorit dimulai... ===" -ForegroundColor Cyan # Daftar aplikasi (ID paket Winget) $apps = @( "MarkText.MarkText", "AIMP.AIMP", "Microsoft.VisualStudioCode", "Notepad++.Notepad++", "VideoLAN.VLC", "7zip.7zip", "Audacity.Audacity", "Discord.Discord", "Spotify.Spotify", "REALiX.HWiNFO", "Microsoft.PowerToys", "BlastApps.FluentSearch", "Git.Git", "Python.Python.3", "Zen-Team.Zen-Browser", "OBSProject.OBSStudio", "Notion.Notion", "SlackTechnologies.Slack", "Zoom.Zoom", "Valve.Steam" ) # Loop instalasi foreach ($app in $apps) { Write-Host ">> Menginstal $app ..." -ForegroundColor Yellow winget install --id $app --silent --accept-package-agreements --accept-source-agreements } Write-Host "=== Semua aplikasi sudah diproses! ===" -ForegroundColor Green
 

Hemat waktu banget

 

Instal manual 20 aplikasi bisa makan 2–3 jam. Dengan Winget, cukup 10–15 menit. Kamu juga bisa bikin skrip pribadi (batch/PowerShell) berisi daftar aplikasi favorit, jadi setiap kali setup ulang Windows tinggal jalankan skrip itu.

 
👉 Intinya: Winget bikin setup Windows jadi cepat, aman, dan otomatis. Sekali coba, rasanya malas balik ke cara manual lagi.
