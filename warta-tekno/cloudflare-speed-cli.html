Fitur-Fitur Mantap yang Bikin Ketagihan

  • **Tes Kecepatan Lengkap**: Ukur download/upload throughput, latency saat idle, dan latency pas lagi beban berat.
  • **TUI Interaktif**: Chart dan statistik muncul real-time, nggak statis kayak speedtest-cli biasa.
  • **Riwayat Tes**: Simpan dan lihat hasil tes sebelumnya, bahkan bisa di-manage.
  • **Ekspor ke JSON**: Buat yang suka analisis data atau scripting.
  • **Mode Text/JSON**: Bisa dijalanin headless, cocok buat cron job atau script otomatis.
  • **Binding Interface**: Pilih network interface spesifik atau source IP, berguna banget kalau punya banyak koneksi.

Sumpah, fitur-fiturnya bikin tool biasa kayak speedtest.net atau fast.com jadi keliatan kuno. Ini langsung pake endpoint resmi Cloudflare, jadi hasilnya akurat dan cepet! ğŸŒ©ï¸

Cara Instalasi Gampang Banget

Ada beberapa cara, pilih yang paling kamu suka:

1. Dari Source (Rekomendasi kalau udah punya Cargo)

cargo install --git https://github.com/kavehtehrani/cloudflare-speed-cli --features tui

2. Pake Script Instalasi (Buat yang Males Ngetik Panjang)

curl -fsSL https://raw.githubusercontent.com/kavehtehrani/cloudflare-speed-cli/main/install.sh | sh

3. Download Binary Langsung

Ambil file binary statis dari rilis terbaru di GitHub. Udah dites di Linux x64/ARM64 dan Windows 11. Kalau pake Mac dan ada masalah, langsung lapor ke dev-nya ya!

Cara Pakai Sehari-hari

Default langsung TUI keren:

cloudflare-speed-cli

Kalau mau output teks biasa (buat scripting):

cloudflare-speed-cli --text

Lihat semua opsi:

cloudflare-speed-cli --help
