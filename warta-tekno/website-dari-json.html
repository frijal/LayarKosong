Membuat Website dari Data .json? Tentu Saja Bisa!

Tentu saja bisa. Membuat website yang datanya berasal dari file `.json` bukan hanya mungkin, tetapi juga merupakan **praktik standar dan sangat umum** dalam pengembangan web modern.

Konsep ini dikenal sebagai "Data-Driven Website," di mana tampilan (desain) dan konten (data) dipisahkan. Ini adalah salah satu konsep inti yang membuat *framework* modern seperti React, Vue, dan Angular sangat populer.

Bagaimana Cara Kerjanya?

Prosesnya secara umum mengikuti tiga langkah utama. Mari kita buat contoh blog sederhana.

Langkah 1: Siapkan Data dalam Format JSON

Pertama, Anda siapkan semua konten Anda dalam sebuah file, misalnya `data.json`. JSON (JavaScript Object Notation) menggunakan format kunci-nilai (*key-value*) yang mudah dibaca manusia dan mesin.


{
  "posts": [
    {
      "id": 1,
      "title": "Selamat Datang di Website Saya",
      "author": "Frijal",
      "date": "2025-10-23",
      "content": "Ini adalah postingan pertama di website yang dibuat sepenuhnya dari data JSON. Sangat fleksibel!"
    },
    {
      "id": 2,
      "title": "Mengapa JSON Sangat Berguna?",
      "author": "Frijal",
      "date": "2025-10-24",
      "content": "Karena JSON memisahkan data dari tampilan, kita bisa mengubah desain tanpa merusak konten."
    }
  ]
}

Langkah 2: Buat "Cetakan" di HTML

Selanjutnya, Anda siapkan "wadah" atau "cetakan" kosong di file `index.html` Anda. Di sinilah JavaScript akan "menuangkan" data nanti.


<!DOCTYPE html>
<html lang="id">
<head>
    <title>Blog dari JSON</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <header>
        <h1>Blog Saya</h1>
    </header>
    <main id="blog-posts-container">
        <!-- Konten dari JSON akan ditampilkan di sini -->
    </main>
    <script src="script.js"></script>
</body>
</html>

Perhatikan `div` dengan id="blog-posts-container". Saat ini, bagian tersebut kosong.

Langkah 3: Gunakan JavaScript untuk Menghubungkan Keduanya üîå

Ini adalah langkah intinya. Dengan JavaScript (kita sebut `script.js`), Anda akan membaca file `.json`, mengolah datanya, dan mengubahnya menjadi elemen HTML.


// Menunggu hingga semua elemen HTML dimuat
document.addEventListener('DOMContentLoaded', () => {
    // Ambil elemen wadah dari HTML
    const container = document.getElementById('blog-posts-container');
    // Gunakan 'fetch' untuk mengambil data dari file data.json
    // 'fetch' adalah cara modern JS untuk meminta data
    fetch('data.json')
        .then(response => response.json()) // Ubah respons teks menjadi format JSON yang bisa dibaca JS
        .then(data => {
            // 'data' sekarang berisi semua objek dari file .json
            let htmlContent = ''; // Siapkan variabel string kosong untuk menampung HTML
            // Lakukan perulangan (looping) untuk setiap post di dalam data.posts
            data.posts.forEach(post => {
                // Buat string HTML untuk setiap post menggunakan template literal (backtick ``)
                // Ini seperti "mencetak" data ke dalam cetakan HTML
                htmlContent += `
                    <article class="post">
                        <h2>${post.title}</h2>
                        <p class="meta">Ditulis oleh ${post.author} pada ${post.date}</p>
                        <p>${post.content}</p>
                    </article>
                `;
            });
            // Setelah semua HTML terkumpul, masukkan ke dalam wadah di halaman web
            container.innerHTML = htmlContent;
        })
        .catch(error => {
            // Penanganan jika file .json gagal dimuat
            console.error('Error memuat data:', error);
            container.innerHTML = '<p>Maaf, gagal memuat artikel.</p>';
        });
});

Hasilnya, halaman web Anda akan secara dinamis menampilkan dua artikel yang berasal dari file `data.json`.

Mengapa Pendekatan Ini Keren Banget?

  • **Pemisahan Logika**: Anda bisa mengubah total desain (HTML/CSS) tanpa menyentuh satu baris pun konten di `.json`.
  • **Manajemen Konten Mudah**: Untuk menambah artikel baru, Anda hanya perlu mengedit file `.json` yang sederhana, tidak perlu menyentuh kode HTML yang rumit.
  • **Dinamis & Skalabel**: Anda bisa memiliki ribuan item di file `.json`, dan JavaScript akan secara otomatis membuatkan halamannya untuk Anda.
  • **Satu Sumber Data**: File `.json` yang sama bisa digunakan untuk website, aplikasi mobile (Android/iOS), atau layanan lainnya, memastikan konsistensi data di semua platform.

Jadi, ya, tidak hanya bisa, tapi ini adalah cara yang sangat direkomendasikan untuk membangun web di era modern!
