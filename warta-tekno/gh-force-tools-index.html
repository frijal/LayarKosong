⚙️ gh-force-tools: Otomatisasi GitHub Dev

Paket dua skrip andalan untuk mengelola branch GitHub secara efisien: gh-force-merge.sh & gh-force-clean.sh

🚀 gh-force-merge.sh

Skrip otomatis untuk mendeteksi PR aktif dari branch lokal, lalu mencoba merge via gh. Jika merge gagal, otomatis fallback ke --admin --squash.

🧹 gh-force-clean.sh

Skrip bash untuk membersihkan branch lokal dan remote yang sudah di-merge ke branch utama. Dukungan penuh untuk --dry-run, --auto, dan --base.
