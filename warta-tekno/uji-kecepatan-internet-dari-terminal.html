Uji Kecepatan Internet dari Terminal

Penasaran dengan kecepatan internet Anda tanpa harus membuka browser? Ada cara keren untuk melakukannya langsung dari terminal menggunakan `speedtest-cli`.

Cara Pemasangan `speedtest-cli`

Di Linux (Debian/Ubuntu)

sudo apt update && sudo apt install speedtest-cli

Di Linux (Fedora/CentOS)

sudo dnf install speedtest-cli

Di Windows (via Python)

Pastikan Python sudah terinstal, lalu jalankan perintah ini di PowerShell atau CMD:

pip install speedtest-cli

Mulai Uji Kecepatan

Setelah terpasang, cukup ketik perintah berikut di terminal:

speedtest-cli

Hasilnya akan langsung ditampilkan, lengkap dengan informasi ping, kecepatan unduh (download), dan kecepatan unggah (upload).

Fitur Tambahan yang Keren

  • **Memilih Server Tertentu:** Gunakan `speedtest-cli --list` untuk melihat daftar server, lalu `speedtest-cli --server [ID]` untuk menguji ke server spesifik.
  • **Hasil dalam Bytes:** Gunakan opsi `--bytes` untuk menampilkan hasil dalam MB/s, bukan Mbit/s.
  • **Hasil Berbentuk JSON:** Tambahkan opsi `--json` untuk output yang mudah diolah oleh skrip.
