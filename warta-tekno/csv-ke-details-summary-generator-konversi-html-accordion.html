Generator CSV ke <details> / <summary>

Tempel CSV atau pilih file .csv, lalu generate accordion HTML bertingkat untuk struktur DPP, DPD, DPC, PAC, dan PR. Semua proses berjalan lokal di browser, tanpa upload ke server.

Boleh juga pakai CSV hasil copy dari spreadsheet. Parser mendukung koma di dalam tanda kutip, misalnya "Paijo Nyunggi B.T., S.Pd., S.H.".

Preview akan muncul setelah tombol Generate ditekan.
