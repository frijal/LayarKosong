Refresh GitHub Codespaces dari browser

Rebuild, full rebuild, restart, atau stop-start—kapan pakai yang mana. Artikel ini menjelaskan langkah-langkah di VS Code web, plus diagram alur singkat biar gampang diingat.

Cara Rebuild Codespace dari Browser

  1. Buka repositori di GitHub → tab **Codespaces** → pilih codespace.
  2. Codespace terbuka di browser (VS Code web).
  3. Tekan Ctrl+Shift+P / Cmd+Shift+P → ketik *Rebuild*.
  4. Pilih **Rebuild Container** (cepat, pakai cache) atau **Full Rebuild Container** (bersih total, lebih lama).

Perbedaan Rebuild vs Full Rebuild

ModeApa yang dilakukanKapan dipakai
RebuildPakai cache image lamaPerubahan kecil, refresh ringan
Full RebuildHapus cache, build ulang dari nolError aneh, dependency rusak, ubah konfigurasi besar

Tips Praktis

  • Mulai dari **Restart** dulu, paling cepat.
  • Kalau masih bermasalah, coba **Rebuild**.
  • Kalau tetap error, baru **Full Rebuild**.
