Command warp-cli register Salah? Ini yang Bener + Cara Kamu Sukses!

Update Desember 2025 buat temen-temen di Balikpapan yang lagi setup WARP biar GitHub ngebut 🚀

Eh pembaca, maaf sangat ya tadi gue kasih command salah. 😅 “warp-cli register” emang udah tidak dipake lagi di versi Cloudflare WARP terbaru (per Desember 2025). Errornya langsung bilang unrecognized subcommand dan kasih tip: similar subcommand exists: 'registration'. Nah, yang bener sekarang warp-cli registration new – langsung muncul notice privacy policy dari Cloudflare, minta konfirmasi Accept Terms [y/N].

Kamu udah coba sendiri dan langsung berhasil: ketik y → Success → lanjut warp-cli connect → Success → warp-cli status → Connected + Network healthy! Mantap pembaca, ini artinya WARP udah aktif full, semua traffic kamu lewat Cloudflare, routing MyRepublic yang jelek ke GitHub langsung dibypass. Loading repo, commit, pull request semoga bisa laju dari Balikpapan 🔥

Kenapa Command Lama Error?

Cloudflare update CLI-nya diam-diam. Dulu register langsung, sekarang dipindah ke subcommand registration biar lebih rapi. Kalau ketik warp-cli register, dia kasih pesan error + tip registration. Kamu langsung coba warp-cli registration dan ketemu subcommand new – itu yang kita butuhin buat daftar client pertama kali.

Notice privacy-nya penting dibaca: Cloudflare cuma kumpulin data DNS & traffic minimal (tanpa payload), tidak dijual, sesuai Privacy Policy mereka. Aman kok buat sehari-hari.

Langkah Lengkap yang Kamu Jalankan (Copy-Paste Aja!)

# 1. Daftar client baru (ini yang bener!)
warp-cli registration new
# Muncul notice privacy → ketik y lalu Enter
# Output: Success
# 2. Connect ke WARP
warp-cli connect
# Output: Success
# 3. Cek status (pastikan Connected!)
warp-cli status
# Output: Status update: Connected
# Network: healthy

Kalau mau disconnect kapan aja tinggal warp-cli disconnect. Mau ubah mode (misal DNS-only): warp-cli mode doh. Atau cek detail registration: warp-cli registration show.

Tips tambahan: Kalau pertama kali connect lambat 10-20 detik, normal – lagi handshake. Setelah itu stabil sangat. Mau auto-start pas boot? sudo systemctl enable --now warp-svc.
