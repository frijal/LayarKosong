Cara Menghilangkan Recommended dan All Apps di Start Menu Windows 11: Panduan Praktis dan Teknis

Microsoft akhirnya memberikan desain Start Menu yang lebih bersih dan terpusat di Windows 11. Analisis Strategis: Prinsip 5W+1H

  • **What (Apa):** Modifikasi antarmuka Start Menu Windows 11 untuk menyembunyikan elemen "Recommended" dan tombol "All Apps".
  • **Why (Kenapa):** Untuk mengurangi gangguan visual, meningkatkan privasi (karena file baru tidak langsung terpampang), dan memaksimalkan *real estate* layar untuk aplikasi kerja utama.
  • **Who (Siapa):** Pengguna rumahan, Power User, Sysadmin, dan IT Manager yang mengelola armada PC di kantor.
  • **When (Kapan):** Dilakukan saat ingin melakukan "debloating" sistem atau setelah instalasi bersih (clean install) Windows 11.
  • **Where (Dimana):** Pengaturan dilakukan langsung di dalam sistem operasi melalui editor kebijakan, registri, atau konsol manajemen.
  • **How (Bagaimana):** Menggunakan teknik mulai dari GUI (Settings) hingga skrip otomatisasi tingkat lanjut.

Latar Belakang dan Evolusi Start Menu

Ketika Windows 11 diluncurkan, Microsoft membagi Start Menu menjadi area tersemat dan area Recommended. Area Recommended ini secara otomatis menampilkan file yang baru saja dibuka, aplikasi baru, dan saran dari Microsoft. Meskipun tujuannya adalah memudahkan akses, bagi banyak orang, ini justru memakan ruang vertikal yang signifikan.

Administrator dan pengguna berpengalaman segera menemukan bahwa kebijakan Group Policy dan kunci registri tertentu dapat menyembunyikan blok Recommended ini sepenuhnya. Microsoft sendiri mendokumentasikan kebijakan `HideRecommendedSection` yang ditujukan untuk perangkat yang dikelola. Namun, seiring berjalannya waktu dan berbagai pembaruan Windows, perilaku kebijakan ini terus berevolusi.

**Kenapa ini penting sekarang?** Karena Microsoft terus melakukan iterasi pada pengalaman Start Menu. Pada build terbaru, Microsoft mengerjakan ulang tata letak Startâ€”termasuk tata letak satu kolom yang dapat digulir secara vertikalâ€”yang mungkin mengubah cara kerja tweak registri lama. Oleh karena itu, kamu harus memperlakukan setiap modifikasi ini sebagai hal yang sensitif terhadap versi sistem operasi yang kamu gunakan.

Pilihan Cepat Sekilas Mata

  1. **Gunakan Settings (Mudah):** Mematikan konten yang muncul, tapi biasanya tetap menyisakan *header* kosong.
  2. **Gunakan Group Policy (Pro/Enterprise):** Kontrol resmi untuk menghapus blok Recommended dan daftar All Apps.
  3. **Registry Edit (Semua Edisi):** Solusi bagi pengguna edisi Home untuk mendapatkan fungsionalitas yang sama dengan Group Policy.
  4. **PowerShell (Otomatisasi):** Cara tercepat untuk menerapkan perubahan melalui skrip, ideal untuk admin.
  5. **MDM/CSP (Enterprise):** Menggunakan Intune untuk menerapkan pengaturan di ribuan PC sekaligus.

*Peringatan: Selalu buat System Restore Point sebelum melakukan modifikasi sistem!*

Metode 1: Lewat Menu Settings (Cara Resmi dan Paling Aman)

Jika kamu lebih suka pendekatan grafis dan ingin mencoba kontrol resmi terlebih dahulu, ini adalah langkah awal yang tepat. Meskipun cara ini tidak selalu menghapus *ruang* yang digunakan oleh Recommended, ia akan menghapus *kontennya*.

Langkah-langkah:

Buka **Settings** â†’ **Personalization** â†’ **Start**, lalu matikan opsi berikut:

  • Show recently added apps
  • Show recommended files in Start
  • Show suggestions for tips, shortcuts, new apps, and more
  • Show websites from your browser history

Melakukan hal ini akan menghapus semua file terbaru dan saran yang memenuhi daftar. Di banyak build terbaru, bagian Recommended akan menciut, namun judul (header) "Recommended" mungkin masih terlihat. Ini adalah opsi yang paling aman karena tidak berisiko merusak stabilitas sistem.

Metode 2: Group Policy Editor (Untuk Pro, Enterprise, dan Education)

Group Policy (GP) menyediakan kontrol "resmi" yang paling jelas, namun memiliki beberapa catatan penting (caveat).

Cara Menggunakan:

  1. Tekan `Win + R`, ketik `gpedit.msc`, dan tekan Enter.
  2. Navigasi ke: **Computer Configuration** (atau User Configuration) â†’ **Administrative Templates** â†’ **Start Menu and Taskbar**.
  3. Untuk menghapus Recommended: Cari dan aktifkan **"Remove Recommended section from Start Menu"**.
  4. Untuk menghapus daftar All Apps: Cari dan aktifkan **"Remove All Programs list from the Start menu"**.

Setelah mengaktifkan kebijakan ini, jalankan perintah `gpupdate /force` di Command Prompt atau lakukan *restart* explorer.exe untuk melihat hasilnya.

**Catatan Praktis:** Group Policy tidak tersedia secara bawaan di Windows 11 Home. Selain itu, Microsoft terkadang membatasi perilaku kebijakan ini hanya untuk SKU tertentu seperti edisi SE atau Education di versi build tertentu.

Metode 3: Registry Edits (Untuk Pengguna Home dan Otomatisasi)

Jika kamu menggunakan Windows 11 Home atau ingin membuat skrip perubahan, kamu bisa membuat nilai DWORD di registri berikut ini.

Menyembunyikan Bagian Recommended:

Key: HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\Explorer
Value: HideRecommendedSection (DWORD)
Data: 1 (untuk menyembunyikan), 0 (untuk menampilkan)

Menghapus Daftar All Apps:

Key: HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer
Value: NoStartMenuMorePrograms (DWORD)
Data: 1 (untuk menghapus), 0 (untuk memulihkan)

Setelah mengatur nilai ini, kamu perlu keluar (sign out) dan masuk kembali, atau me-restart proses Windows Explorer melalui Task Manager agar perubahan terasa efeknya.

Skrip PowerShell (Jalankan sebagai Administrator):

# Set HideRecommendedSection
New-ItemProperty -Path "HKLM:\SOFTWARE\Policies\Microsoft\Windows\Explorer" -Name "HideRecommendedSection" -PropertyType DWord -Value 1 -Force

# Set NoStartMenuMorePrograms
New-ItemProperty -Path "HKCU:\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer" -Name "NoStartMenuMorePrograms" -PropertyType DWord -Value 1 -Force

Apa yang Diharapkan Setelah Perubahan?

Kita harus menekankan bahwa hasil modifikasi ini bisa bervariasi. Pada beberapa build Windows 11, bagian Recommended benar-benar hilang sehingga area Pinned Apps bisa meluas memenuhi Start Menu. Namun pada build lain, area tersebut mungkin hanya menjadi ruang kosong yang tidak bisa digunakan.

Selain itu, Microsoft sering mengubah perilaku UI pada update mingguan. Tweak yang bekerja hari ini mungkin akan tereset atau diabaikan oleh pembaruan fitur di masa depan. Oleh karena itu, selalu simpan dokumentasi perubahan yang kamu buat agar mudah dikembalikan jika terjadi ketidakstabilan.

Troubleshooting: Jika Tidak Berubah

  • Pastikan tipe data adalah **DWORD (32-bit)** meskipun kamu menggunakan sistem 64-bit.
  • Cek apakah ada konflik dengan kebijakan MDM jika PC kamu adalah milik kantor (domain-joined).
  • Beberapa alat keamanan atau antivirus mungkin menandai perubahan registri kebijakan sebagai "Potentially Unwanted Modification" (PUM). Pastikan untuk mengecualikan (whitelist) perubahan tersebut jika kamu yakin.

Alternatif Jika Cara Native Gagal

Jika kebijakan resmi Microsoft masih menyisakan ruang kosong yang mengganggu, ada opsi pihak ketiga yang sangat populer di komunitas power user:

  • **ExplorerPatcher:** Tool open-source yang sangat kuat untuk mengembalikan fungsionalitas taskbar dan Start Menu ke gaya yang lebih klasik atau lebih bersih.
  • **StartAllBack:** Utilitas berbayar yang memberikan kontrol UI sangat mendalam dan performa yang sangat ringan.
  • **Flow Launcher:** Alternatif Start Menu berbasis pencarian yang bisa menggantikan penggunaan Start Menu bawaan sepenuhnya.

*Peringatan: Penggunaan tool pihak ketiga meningkatkan risiko kompatibilitas saat Windows Update besar dirilis. Gunakan dengan bijak.*

Kesimpulan dan Rekomendasi Akhir

Mencapai Start Menu yang minimalis di Windows 11 memerlukan sedikit usaha lebih karena filosofi desain Microsoft yang berfokus pada konten rekomendasi. Bagi sebagian besar pengguna, menonaktifkan konten di menu Settings sudah cukup. Namun bagi kamu yang mengejar estetika *pixel-perfect* dan efisiensi ruang, metode Registry dan Group Policy adalah jalan yang harus ditempuh.

Selalu lakukan pengujian pada satu perangkat sebelum menerapkan skrip ini ke seluruh komputer di kantor. Dokumentasikan setiap langkah, dan jangan lupa untuk selalu menyiapkan jalan pulang (rollback plan) dengan System Restore. Selamat berkreasi dengan desktop kamu!
