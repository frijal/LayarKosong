🔐 Cara Membuat API di Cloudflare

Cloudflare menyediakan API yang memungkinkan kamu mengelola DNS, konfigurasi keamanan, cache, dan banyak fitur lainnya secara dapat — tanpa perlu membuka dashboard. API ini sangat berguna kalau kamu ingin menghubungkan website, aplikasi, atau sistem otomatis (seperti CI/CD GitHub Actions).

Langkah 1: Masuk ke Akun Cloudflare

Buka Cloudflare Dashboard dan login ke akun kamu. Setelah masuk, klik ikon profil di kanan atas, lalu pilih “My Profile”.

Langkah 2: Buat API Token

Masuk ke tab API Tokens, lalu klik tombol “Create Token”. Pilih template “Edit zone DNS” jika kamu hanya ingin mengelola DNS, atau buat kustom sesuai kebutuhan. Setelah selesai, salin token tersebut dan simpan di tempat aman.

⚠️ Catatan: Token API tidak bisa ditampilkan ulang setelah dibuat, jadi pastikan kamu menyimpannya!

Langkah 3: Gunakan API Key

Selain API Token, Cloudflare juga punya Global API Key — ini seperti “kunci utama” akun kamu. Kamu bisa menemukannya di halaman yang sama, di bagian “API Keys”. Gunakan dengan hati-hati, karena ini memberi akses penuh ke akunmu.

Langkah 4: Uji API Menggunakan curl

Setelah punya token, kamu bisa mencoba memanggil API Cloudflare lewat terminal, misalnya:

curl -X GET "https://api.cloudflare.com/client/v4/user" \
-H "Authorization: Bearer YOUR_API_TOKEN" \
-H "Content-Type: application/json"

Kalau berhasil, kamu akan melihat data akunmu dalam format JSON.

Penutup

Dengan API Cloudflare, kamu bisa mengotomatisasi banyak hal, seperti update DNS otomatis, flush cache, atau pengaturan keamanan. Ini langkah penting buat kamu yang ingin situsmu lebih efisien dan modern.
