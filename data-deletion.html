Instruksi Penghapusan Data

Kami menghormati privasi Anda. Berikut cara meminta penghapusan data yang terkait dengan layanan kami.

Yang dapat diminta dihapus

  • Komentar atau kontribusi yang Anda kirim
  • Data profil (nama tampilan, email)
  • Data teknis sesi yang dapat diidentifikasi

Cara mengajukan permintaan (3 opsi)

  1. Email
    Kirim ke sini@dalam.web.id
    Subjek: Permintaan Penghapusan Data
    Sertakan: nama tampilan, email, URL/ID konten, dan keterangan singkat.
  2. Formulir web
    https://dalam.web.id/data-deletion-form
  3. Endpoint API (machine-readable)
    POST https://dalam.web.id/api/data-deletion-request
Content-Type: application/json

{
  "identifier": "user@example.com",
  "type": "email|comment_id|url",
  "reason": "User requested deletion"
}
    Contoh respon:
    { "status":"received", "ticket":"DD-2025-0001", "estimated_completion":"2025-12-08T12:00:00Z" }

Proses & waktu

Respon dalam 24–72 jam kerja. Kasus kompleks akan kami beri estimasi tambahan via email.

Verifikasi

Kami mungkin meminta bukti kepemilikan (email konfirmasi, screenshot, dll).

Data pihak ketiga

Komentar via Facebook/Google dkk. tetap harus dihapus melalui platform masing-masing. Kami akan membantu mengarahkan.

Pertanyaan lebih lanjut: sini@dalam.web.id

Data Deletion Instructions

We respect your privacy. Here’s how to request deletion of your data.

What can be deleted

  • Comments or contributions you submitted
  • Profile data (display name, email)
  • Identifiable technical/session data

How to request (3 options)

  1. Email
    Send to sini@dalam.web.id
    Subject: Data Deletion Request
    Include: display name, email, URL/ID of content, short description.
  2. Web form
    https://dalam.web.id/data-deletion-form
  3. API endpoint (machine-readable)
    POST https://dalam.web.id/api/data-deletion-request
Content-Type: application/json

{
  "identifier": "user@example.com",
  "type": "email|comment_id|url",
  "reason": "User requested deletion"
}
    Example response:
    { "status":"received", "ticket":"DD-2025-0001", "estimated_completion":"2025-12-08T12:00:00Z" }

Process & timeline

Response within 24–72 business hours. Complex cases will receive an updated estimate via email.

Verification

We may ask for proof of ownership (confirmation email, screenshots, etc.).

Third-party data

Comments made via Facebook/Google etc. must be deleted on those platforms. We will assist where possible.

Questions: sini@dalam.web.id
