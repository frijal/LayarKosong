XPenguins: Penguin lucu jatuh dari atas layar

Ingat paket kecil bernama XPenguins? Ia menebar penguin yang terjun bebas dari tepi layar, mendarat, lalu beraksi kocak. Cocok untuk menemani waktu tunggu di depan komputer.

XPenguins Linux Fun Animasi
Apa itu XPenguins?

XPenguins adalah aplikasi ringan yang menampilkan karakter penguin jatuh dari atas layar, berjalan, melompat, hingga berinteraksi dengan jendela di desktop X11. Aplikasi ini murni hiburan: tidak mengganggu kerja, tetapi menyenangkan untuk dilihat sambil menunggu.

Video demo di atas adalah hasil rekaman XPenguins yang bisa langsung ditonton.

Instalasi cepat

Debian/Ubuntu/BlankOn:

sudo apt-get update
sudo apt-get install xpenguins

Fedora:

sudo dnf install xpenguins

Arch/Manjaro:

sudo pacman -S xpenguins

Nama paket dapat bervariasi pada rilis tertentu. Jika tidak tersedia di repo resmi, cek AUR/PPAs atau kompilasi dari sumber.

Cara pakai dan opsi populer
  1. Menjalankan dasar: buka terminal, jalankan xpenguins, biarkan animasi berjalan di atas desktop Anda.
  2. Memilih tema: gunakan parameter -t untuk memilih tema penguin jika tersedia.
  3. Mengatur jumlah: pakai -n untuk mengatur banyaknya penguin agar tetap ringan.
  4. Menutup: tekan Ctrl + C di terminal tempat XPenguins dijalankan.

Contoh perintah

# Menjalankan dengan jumlah penguin tertentu
xpenguins -n 30
# Menjalankan dengan tema (jika tema tersedia pada sistem)
xpenguins -t "Penguins"
# Menjalankan di layar tertentu (multi-monitor, X11)
DISPLAY=:0 xpenguins -n 20

Catatan: XPenguins berjalan di lingkungan X11 tradisional. Pada Wayland, kompatibilitas bisa terbatas dan memerlukan XWayland.
