Layar Kosong Bookmark Link Manager

Kelola daftar URL dan simpan ke dalam database JSON

Cara Kerja:

  1. Masukkan URL ‚Üí Klik tombol Crawl ‚Üí Data hasil crawling akan muncul di antrean.
  2. Klik Export to JSON ‚Üí Simpan file sebagai data.json di folder yang sama.
  3. Saat halaman dibuka, sistem otomatis membaca file data.json (jika tersedia) dan menampilkan hasilnya lengkap dengan fitur Search dan Sort.

Karena JavaScript di browser tidak diizinkan menulis atau membuat file langsung ke folder server (demi alasan keamanan), maka digunakan mekanisme Download JSON sebagai solusinya. Cara ini disebut Pre-fetching. Intinya, kita merayap di awal untuk mengambil data (crawl), lalu menyimpan hasilnya ke dalam file JSON. Setelah itu, browser hanya perlu membaca file JSON yang ringan tersebut. Hasilnya? Proses jadi jauh lebih cepat dan lebih hemat kuota.
