Instal di Ubuntu & Debian (Termasuk Debian 13 Trixie)

Cara resmi, pakai $(lsb_release -cs) otomatis ‚Äì sekarang Trixie langsung support!

# Add cloudflare gpg key (WAJIB re-run kalau instal sebelum Sept 2025!)
curl -fsSL https://pkg.cloudflareclient.com/pubkey.gpg | sudo gpg --yes --dearmor --output /usr/share/keyrings/cloudflare-warp-archive-keyring.gpg
# Add repo
echo "deb [signed-by=/usr/share/keyrings/cloudflare-warp-archive-keyring.gpg] https://pkg.cloudflareclient.com/ $(lsb_release -cs) main" | sudo tee /etc/apt/sources.list.d/cloudflare-client.list
# Install
sudo apt-get update && sudo apt-get install cloudflare-warp

Kalau masih error locate package: Clear cache apt (sudo apt clean), atau cek codename kamu bener (lsb_release -cs harus trixie/bookworm dll).

Instal di Red Hat Enterprise Linux & CentOS (Versi 8)

# Update public key (kalau perlu)
sudo rpm -e 'gpg-pubkey(4fa1c3ba-61abda35)' && sudo rpm --import https://pkg.cloudflareclient.com/pubkey.gpg
# Add repo
curl -fsSl https://pkg.cloudflareclient.com/cloudflare-warp-ascii.repo | sudo tee /etc/yum.repos.d/cloudflare-warp.repo
# Update & Install
sudo yum update
sudo yum install cloudflare-warp

Setelah Instal, Aktifin WARP

warp-cli registration new
warp-cli connect

Cek status: warp-cli status ‚Äì harus Connected! Disconnect kalau mau: warp-cli disconnect.
