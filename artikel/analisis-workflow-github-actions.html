Di sini proses "sulap" file terjadi. Intip langkah kuncinya:

Sinkronisasi File

Pake rsync dengan aturan whitelist yang ketat.

rsync -a --delete \
  --include='artikel/***' \
  --include='ext/***' \
  --exclude='*' \
  source/ deploy_dir/

Deteksi Perubahan

Bandingin folder baru sama yang udah live. Kalau isinya sama, ya nggak usah repot-repot deploy!

if git diff --no-index --quiet deploy_dir/ site/; then
  echo "changed=false" >> $GITHUB_OUTPUT
fi
