Cache pip bisa membengkak seiring waktu. Supaya lebih praktis, kita bisa bikin skrip shell sederhana untuk membersihkannya secara otomatis.

Simpan kode berikut sebagai bersih-pip-cache.sh:

echo "üßπ Membersihkan cache pip..."

if ! command -v pip &> /dev/null; then
  echo "‚ùå pip tidak ditemukan. Pastikan Python dan pip sudah terinstal."
  exit 1
fi

echo "üì¶ Info cache pip sebelum dibersihkan:"
