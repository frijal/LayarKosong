Generator Tabel PDF untuk WordPress Classic

Masukkan banyak URL/path PDF, lalu generator ini akan membuat kode HTML tabel responsif berisi nama dokumen, tombol Baca, dan tombol Unduh. Semua proses berjalan lokal di browser.

Input Data PDF

1. Paste URL/path banyak PDF

Format baris yang didukung: URL saja, Judul | URL, Judul | Keterangan | URL, atau No | Judul | Keterangan | URL.

2. Import dari file

File .txt bisa berisi daftar URL per baris. File .csv mendukung kolom seperti no,title,note,url atau nomor,nama,keterangan,link.

3. Ambil nama dari banyak file PDF lokal

Karena aturan keamanan browser, input ini tidak bisa membaca path folder asli. Generator akan memakai nama file dan menggabungkannya dengan Base URL di bawah.

4. Pengaturan URL dan judul

Dipakai untuk file PDF lokal atau baris yang hanya berisi nama file.

5. Tambah baris manual

6. Opsi output

Format cepat

  • URL → judul dibuat otomatis dari nama file.
  • Judul | URL → keterangan kosong.
  • Judul | Keterangan | URL → format paling enak untuk WordPress.
  • No | Judul | Keterangan | URL → nomor manual ikut dipakai.
