Cara Membuat API Cloudflare

Pernah ingin otomatis hapus cache, ubah DNS, atau deploy halaman tanpa buka Cloudflare manual? Nah, kamu bisa melakukannya lewat Cloudflare API Token. Berikut panduan lengkapnya.

Langkah 1 ‚Äì Masuk ke Halaman API Tokens

Buka situs: https://dash.cloudflare.com/profile/api-tokens Lalu klik tombol ‚ÄúCreate Token‚Äù.

Langkah 2 ‚Äì Pilih Jenis Token

Kamu bisa pilih template yang sudah disediakan Cloudflare (lebih aman), misalnya ‚ÄúZone: Cache Purge‚Äù, atau bikin token custom sesuai kebutuhan.

Permissions:
- Zone ‚Üí Cache Purge ‚Üí Edit
- Zone ‚Üí Zone ‚Üí Read

Kalau mau juga ubah DNS otomatis:

- Zone ‚Üí DNS ‚Üí Edit

Langkah 3 ‚Äì Buat dan Simpan Token

Setelah klik ‚ÄúCreate Token‚Äù, Cloudflare akan menampilkan kode API seperti ini:

Salin dan simpan baik-baik ‚Äî token ini cuma muncul satu kali!

Langkah 4 ‚Äì Simpan Token di GitHub Secrets

Kalau kamu pakai GitHub Actions, buka:

Settings ‚Üí Secrets and variables ‚Üí Actions ‚Üí New repository secret

Lalu isi:
Name: CF_API_TOKEN
Value: tempel token kamu

     -H "Authorization: Bearer <TOKEN_KAMU>" \
     -H "Content-Type: application/json"
