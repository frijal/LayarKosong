📦 Hanya Pasang yang Kamu Butuhkan (Minimal Install)

Salah satu masukan paling sering dari komunitas adalah ingin memulai dengan basis yang ringan, lalu menambahkan aplikasi sesuai kebutuhan. Mulai Ubuntu Studio 24.10, opsi minimal install tersedia di installer! Kamu bisa memilih instalasi dasar dengan tema Ubuntu Studio, konfigurasi audio yang optimal, tetapi tanpa semua aplikasi kreatif. Selanjutnya, pakai Ubuntu Studio Installer untuk menambahkan workflow yang kamu mau: audio, grafis, video, fotografi, atau publishing — ala carte! 🍽️

Jika kamu sudah menjalankan Kubuntu, Ubuntu biasa, atau flavour resmi lainnya, kamu tidak perlu instal ulang. Cukup install paket Ubuntu Studio Installer dan pilih komponen yang diinginkan. Opsi full install juga tetap tersedia buat yang ingin workstation kreatif all-in-one.

✨ Fitur-Fitur Baru di Rilis Ini

Meskipun ini rilis LTS yang mengutamakan stabilitas, banyak peningkatan signifikan sejak 24.04 LTS. Berikut yang paling keren:

  • 🎛️ Tiga Tata Letak Desktop: Kini tersedia tiga layout pilihan: panel atas klasik ala Ubuntu Studio, layout mirip macOS (global menu + dock bawah), dan layout baru mirip Windows 10 (panel bawah). Komunitas melalui voting memilih layout bottom panel (tradisional) sebagai default. Cocok buat pengguna baru yang datang dari berbagai ekosistem.
  • 🔄 Ubuntu Studio Installer & Audio Configuration ditulis ulang total: Kini pakai Python dengan backend UI ganda (GTK4 dan Qt6) yang otomatis mendeteksi DE-mu. Dukungan FFADO untuk audio FireWire, konfigurasi buffer PipeWire via dropdown, serta terjemahan 21 bahasa.
  • 🔊 Tema Suara Borealis: Kembali setelah era Gutsy Gibbon! Nuansa suara yang fresh menggantikan tema Ocean.
  • 🎧 Loopino: Plugin sampler audio ringan (LV2/CLAP/VST2) untuk load, trim, dan looping file audio dengan drag-and-drop serta rekaman langsung.
  • Plasma PipeWire Settings: Applet baru di system tray untuk mengelola konfigurasi PipeWire dengan mudah.
  • Driver snd-hdspe DKMS: Dukungan untuk kartu suara RME HDSPe series (MADI, AES, RayDAT, AIO) tersedia di repositori.
  • DistroAV (ex OBS-NDI): Sekarang tersedia di repositori untuk streaming audio/video jaringan di OBS Studio.
  • PipeWire tetap menjadi server audio default dengan peningkatan performa.

⚠️ Masalah yang Diketahui (Known Issues)

  • Minor kosmetik: saat transisi dari sesi instal ke live desktop, muncul splash KDE Plasma default (bukan Ubuntu Studio). Ini tidak terjadi setelah instalasi selesai dan akan diperbaiki di build berikutnya.
  • Setelah login pertama, kamu akan diminta reboot untuk menerapkan konfigurasi audio produksi. Ini workaround karena installer belum bisa otomatis menambahkan user pertama ke grup audio. (bug #2063899)
  • Catatan rilis resmi Ubuntu Studio: di sini dan rilis Kubuntu: Kubuntu Known Issues

💬 Pertanyaan yang Sering Diajukan (FAQ)

Q: Apakah Ubuntu Studio mengandung paket snap?
A: Iya, karena Mozilla Firefox kini disarankan snap (kesepakatan distribusi), demikian juga Thunderbird demi kecepatan patch keamanan. Hasil dapat berbeda tergantung perangkat, versi paket, dan konfigurasi sistem. FreeShow (Electron) juga disediakan sebagai snap agar bisa masuk repositori. Kami bekerja sama dengan Canonical untuk meningkatkan kualitas snap. Beri kesempatan! 😉

Q: Jika saya install beta ini, apakah perlu instal ulang saat rilis final? Hasil dapat berbeda tergantung perangkat, versi paket, dan konfigurasi sistem.
A: Tidak. Selalu update sistem, maka kamu akan bertransisi ke rilis final secara mulus.

Q: Bisakah saya minta ISO dengan desktop environment favorit saya?
A: Kami tim sukarelawan, tidak punya sumber daya membuat flavour baru. Solusi: pakai flavour resmi (Kubuntu, Xubuntu, dll) lalu tambahkan Ubuntu Studio Installer untuk mendapatkan manfaat Ubuntu Studio tanpa konversi penuh.

Q: Bagaimana jika tidak ingin semua paket terinstall?
A: Gunakan opsi minimal install di awal, lalu pilih workflow lewat Ubuntu Studio Installer sesuai kebutuhan.
