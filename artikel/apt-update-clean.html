Skrip Otomatis Update & Clean APT

Berikut adalah skrip shell sederhana untuk memudahkan proses pembaruan dan pembersihan paket yang tidak terpakai di sistem berbasis Debian/Ubuntu.

⚠️ Catatan: Skrip ini membutuhkan akses sudo. Pastikan Anda menjalankannya dengan hak akses root atau sebagai user dengan izin sudo.

Skrip Lengkap

#!/bin/bash
# Skrip untuk otomatis update & clean sistem APT
echo "=== Memperbarui daftar paket ==="
sudo apt update -y
echo "=== Meng-upgrade paket terpasang ==="
sudo apt upgrade -y
echo "=== Menghapus paket yang tidak diperlukan lagi ==="
sudo apt autoremove -y
echo "=== Membersihkan cache APT ==="
sudo apt autoclean -y
sudo apt clean -y
echo "=== Selesai! Sistem sudah diperbarui dan dibersihkan. ==="
