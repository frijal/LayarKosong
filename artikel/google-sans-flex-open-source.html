Pernah ngerasa mata cepet pegel pas lagi asik ngetik atau ngoding? Seringkali kita nyalahin tingkat kecerahan layar, padahal masalahnya bisa jadi ada di **Font**.

Font itu lebih penting dari yang kita duga. Font yang bagus bisa ngurangin kelelahan mata dan bikin konten di layar jauh lebih gampang dibaca. Kalau di mesin Fedora-ku sih, aku biasanya setia pake font Inter. Tapi ya, selera orang kan beda-beda; ada yang suka yang membulat (rounded), ada yang suka garis tajam nan bersih. Nah, kabar gembiranya, Google baru aja merilis opsi baru yang sudah open source dan wajib banget kamu lirik: Google Sans Flex.

Google Sans Flex: Apa Sih Kelebihannya? ü§î

Dirilis di bawah lisensi SIL Open Font License, Google Sans Flex adalah font generasi terbaru Google yang didesain oleh David Berlow. Istilah kerennya, ini adalah Variable Font.

Apa itu font variabel? Bayangin kamu cuma butuh satu file font, tapi di dalamnya sudah mencakup lima "poros" (axes) yang bisa diatur sesuka hati: ketebalan (weight), lebar (width), ukuran optik, kemiringan, sampai seberapa bulat ujung hurufnya. Jadi, satu file mungil ini bisa menggantikan puluhan file font tradisional yang bikin folder .fonts kamu penuh sesak!

Google bikin font ini khusus buat layar modern dengan berbagai ukuran. Jadi, kalau kamu pake monitor resolusi tinggi (4K) dengan fractional scaling, font ini bakal tetep kelihatan tajam, nggak pecah atau "blur" kayak hubungan kamu sama dia. Haha!

üì• Cara Pasang Google Sans Flex di Linux

Langkahnya gampang banget:

  1. Download file font-nya dari web resmi Google Fonts.
  2. Ekstrak filenya.
  3. Klik dua kali file .ttf-nya lalu klik "Install", atau pindahin ke folder ~/.local/share/fonts/.
  4. Gunakan GNOME Tweaks atau tool bawaan distro kamu buat ganti font sistem.
