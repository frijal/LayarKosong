💻 Cara Gampang Cek Semua Aplikasi Node.js yang Terpasang di Laptop

Panduan cepat pake npm list buat lihat paket global kamu. Nggak pake ribet, langsung kelihatan semua tool CLI yang kamu install.

Dipublikasikan: 27 Okt 2025 npm node.js CLI Tutorial

Hasil dari npm list -g --depth=0, bersih dan rapi.

Seringkali kita install banyak "aplikasi Node.js" tapi lupa apa aja yang udah terpasang. Yang dimaksud "aplikasi" di sini biasanya adalah paket global (global packages), yaitu alat-alat command-line (CLI) yang bisa kamu panggil dari mana aja di terminal.

Contohnya kayak nodemon, pm2, create-react-app, yarn, atau netlify-cli. Nah, cara termudah buat ngeceknya adalah pake perintah bawaan dari NPM sendiri.

1. Mengecek Paket Global (CLI)

Ini cara yang paling umum dan kemungkinan besar yang kamu cari. Buka terminal atau Command Prompt kamu dan ketikkan perintah sakti ini:

npm list -g --depth=0

Penjelasan Perintah:

  • npm list: Ini adalah perintah dasarnya untuk melihat daftar paket.
  • -g: Ini singkatan dari --global. Artinya, kamu spesifik cuma mau cari paket yang terpasang secara global, bukan yang ada di dalam satu folder proyek aja.
  • --depth=0: Nah, ini bagian pentingnya. Perintah ini bilang ke NPM, "Tolong tampilkan paket utamanya aja (depth=0), nggak usah sama semua dependensi-dependensinya."

Kalau kamu nekat menjalankan npm list -g tanpa --depth=0, siap-siap aja terminal kamu banjir teks. Kamu bakal lihat daftar yang super panjang karena semua paket kecil yang dibutuhkan oleh paket utamamu bakal ikut ditampilin.

2. Gimana Kalo Pake Yarn atau PNPM? 🧶

Kalau kamu lebih suka pake package manager lain, perintahnya sedikit beda tapi tujuannya sama.

Untuk Yarn: 
yarn global list

Perintah Yarn ini biasanya udah pinter dan langsung nampilin daftar yang rapi.

Untuk PNPM: 
pnpm list -g --depth=0

Mirip banget sama NPM, pake --depth=0 juga.

3. Penasaran Filenya Disimpen di Mana? 📁

Kalau kamu kepo di mana sebenernya lokasi folder node_modules global kamu disimpen, pake perintah ini:

Lokasi root NPM Global: 
npm root -g

Bonus: Kalo Maksudmu Paket di Dalam Proyek (Lokal)

Ini beda kasus ya. Kalau yang kamu maksud adalah "aplikasi" atau paket yang terpasang *di dalam* satu folder proyek spesifik (yang ada di package.json), caranya gampang:

  1. Masuk dulu ke folder proyeknya:
cd /path/ke/proyek-kamu
  1. Jalankan perintah list tapi tanpa -g:
npm list

Itu bakal nampilin semua paket yang dipake *hanya* untuk proyek itu.
