Install KDE Plasma Desktop di Semua Distro Linux 🐧✨

What (Apa itu KDE Plasma?) — KDE Plasma adalah desktop environment modern, cantik, dan sangat bisa dikustomisasi. Cocok buat kamu yang pengen tampilan ala Windows, macOS, atau bahkan yang benar-benar unik. Ringan? Relatif, tapi performanya makin oke tiap rilis.

Why (Kenapa harus install KDE Plasma?) — Karena kamu bosan dengan GNOME, XFCE, atau desktop bawaan distro. Plasma menawarkan widget, tema global, dan integrasi yang rapi. Plus, dukungan Wayland makin mantap.

Who (Siapa yang butuh panduan ini?) — Pengguna Linux dari level menengah ke atas yang mau ganti DE atau pengen dual DE. Tapi tenang, perintahnya tinggal salin-tempel kok. 😎

Where (Di distro mana aja bisa?) — Di hampir semua distro mainstream: Debian/Ubuntu, Arch, Fedora, openSUSE, Void, Gentoo, bahkan NixOS. Di bawah ini kita kasih perintah lengkap per keluarga distro.

When (Kapan waktu yang tepat?) — Kapan pun kamu merasa desktop sekarang kurang nendang. Atau setelah fresh install server yang pengen disulap jadi workstation. Instalasi tidak sampai 10 menit kok.

How (Bagaimana caranya?) — Pilih distro kamu, buka terminal, jalankan perintah di bawah. Jangan lupa enable SDDM dan reboot. Yuk langsung gas! 🔥

Catatan penting: Pastikan koneksi internet stabil. Perintah menggunakan sudo atau root. Setelah install, enable display manager SDDM agar bisa login ke Plasma.

📋 Daftar Perintah Install KDE Plasma + SDDM

Debian-based Ubuntu, Debian, Mint, Pop!_OS, Kali, dll 
sudo apt update
sudo apt install kde-standard sddm

Untuk varian minimal bisa pakai kde-plasma-desktop.

Arch-based Arch, Manjaro, EndeavourOS, Garuda, CachyOS 
sudo pacman -S plasma sddm

💡 Kalau pakai yay atau AUR helper juga bisa, tapi plasma ada di repo official.

Fedora-based Fedora, Nobara, Ultramarine, Bazzite 
sudo dnf groupinstall "KDE Plasma Workspaces"
RHEL-based RHEL, Alma, Rocky, Oracle, CentOS Stream 
sudo dnf groupinstall "KDE Plasma Workspaces"

⚙️ EPEL mungkin diperlukan, tapi biasanya group KDE sudah tersedia.

openSUSE Leap & Tumbleweed 
sudo zypper install -t pattern kde kde_plasma
Void Linux 
sudo xbps-install -S kde5 sddm
sudo ln -s /etc/sv/sddm /var/service/

🔁 Void pakai runit, jadi symlink ke service.

Gentoo Linux 
sudo emerge --ask kde-plasma/plasma-meta sddm
sudo rc-update add sddm default
Slackware Linux 
slackpkg install kde

KDE sudah include di Slackware (full install)

Alpine Linux 
sudo apk add plasma-desktop sddm
sudo rc-update add sddm
NixOS

Tambahkan ke /etc/nixos/configuration.nix:

services.xserver.enable = true;
services.xserver.displayManager.sddm.enable = true;
services.xserver.desktopManager.plasma5.enable = true;

lalu rebuild

sudo nixos-rebuild switch
Clear Linux 
sudo swupd bundle-add desktop-kde

Enable SDDM & Reboot (untuk systemd)

sudo systemctl enable sddm
reboot

✨ Setelah reboot, di layar login pilih sesi "Plasma" (biasanya otomatis).

Tips Tambahan & Optimasi

👉 Setelah masuk Plasma, kamu bisa install plasma-workspace tambahan, kde-apps, atau plasma-nm untuk network manager. Kalau mau lebih minimal, ganti kde-standard dengan plasma-desktop.
