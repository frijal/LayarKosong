Kombinasi Tombol Pintas Synaptic

Berikut adalah panduan interaktif kombinasi tombol pintas (*shortcut*) untuk Synaptic Package Manager, manajer paket grafis populer di Debian, Ubuntu, dan turunannya.

Daftar Tombol Pintas

Fungsi Tombol Pintas
Perbarui daftar paketCtrl + R
Cari paketCtrl + F
Lihat properti paketCtrl + O
Tandai untuk instalasiCtrl + I
Tandai untuk upgradeCtrl + U
Tandai untuk dihapusDelete
Terapkan perubahanCtrl + P
Keluar dari aplikasiCtrl + Q
