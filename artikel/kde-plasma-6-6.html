Fitur-Fitur Terbaru & Improvements

Customization & Theming yang Lebih Powerful

Simpan Custom Global Theme Sendiri

Salah satu fitur yang paling ditunggu-tunggu adalah kemampuan untuk menyimpan kombinasi color scheme, widgets, dan layouts kamu sebagai global theme yang bisa digunakan kembali atau dibagikan dengan user lain. Ini berarti kamu tidak perlu mengatur ulang desktop dari nol setiap kali install KDE di device baru. Cukup export theme kamu dan import di mana saja!

Gaming & Activity Support yang Lebih Baik

Game Controller Support Upgrade

Ketika kamu menggunakan game controller, sistem akan secara otomatis mencegah screen dari auto-sleeping atau suspend. Ini sangat berguna untuk gaming session yang panjang atau ketika kamu menggunakan media player dengan controller. Tidak perlu khawatir screen tiba-tiba mati di tengah gameplay atau film!

Advanced Security Integration

oo7 Secret Service Provider Support

Fitur baru ini memberikan support yang lebih advanced untuk Secret Service API (oo7), yang berarti integrasi password manager dan keychain menjadi lebih smooth dan secure. Aplikasi yang menggunakan standard secret storage akan memiliki integration yang lebih baik, dan workflow yang lebih aman untuk task-task yang sensitive.

Wayland Improvements

Wayland semakin menjadi default display server untuk Linux modern, dan Plasma 6.6 membawa beberapa improvements penting untuk user yang menggunakan Wayland session:

Slow Keys Support untuk Accessibility

Untuk user yang memiliki motor skill challenges atau kebutuhan khusus lainnya, fitur "Slow Keys" kini tersedia under Wayland. Feature ini memberikan delay antara penekanan tombol dan registrasi input, membantu mencegah unintended keystrokes.

Improved Zoom Effect

Ketika menggunakan magnification feature, pointer kini akan tetap centered di screen, membuat pengalaman zoom menjadi lebih intuitif dan mudah digunakan, terutama untuk user yang membutuhkan screen magnification.

Performance & Display Refresh Rate Fixes

Smooth Animations pada High Refresh Rate Displays

Kalau kamu punya monitor dengan refresh rate lebih dari 60Hz (misalnya 144Hz atau 240Hz yang populer untuk gaming), Plasma 6.6 sekarang support high refresh rate dengan lebih baik. Animasi akan terlihat lebih smooth dan UI akan terasa lebih responsive, yang especially penting untuk gamers dan power users yang menghargai performance details.

Emoji & UI Polish

  • Default Skin Tone untuk Emoji Selector: Kamu bisa pilih default skin tone yang disuka dan semua emoji akan menggunakan setting ini, membuat experience lebih personal dan inclusive.
  • Minor UI Tweaks: Across the board polish untuk membuat interface terasa lebih cohesive dan modern.
  • Widget Improvements: Berbagai widget kecil mendapat polish dan improvement dalam appearance dan functionality.

Quality of Life Polishes

Selain fitur-fitur besar, ada juga beberapa improvement kecil tapi meaningful yang akan membuat daily experience kamu dengan KDE Plasma lebih smooth:

Improved Power Widget

Power widget sekarang menampilkan informasi yang lebih detail tentang aplikasi atau proses apa yang sedang mencegah sleep atau suspend. Ini sangat membantu ketika kamu ingin debug kenapa device tidak sleep atau ketika kamu ingin tahu aplikasi mana yang consuming banyak power.

Bluetooth Tray Fix

Mengelola Bluetooth devices sekarang lebih mudah, terutama fitur untuk "Forget" devices menjadi lebih accessible dan straightforward. Kalau kamu sering ganti device atau ingin remove old pairings, ini akan menghemat klik dan frustration.

Thunderbolt Settings Smart Display

Settings page untuk Thunderbolt sekarang hanya ditampilkan ketika kamu actually punya Thunderbolt hardware. Ini menghilangkan clutter dari settings menu untuk user yang tidak memiliki Thunderbolt ports.

Spectacle Screenshot App Memory

Spectacle (built-in screenshot tool) sekarang mengingat ukuran window dari session sebelumnya. Small improvement, tapi really valuable ketika kamu regularly take screenshots dengan window size yang sama, saves time!

Common Crash Fixes

Tim KDE sudah mengidentifikasi dan fix beberapa crash yang common, terutama yang terjadi ketika disconnect external monitors atau panel-related issues. Update ini akan membuat Plasma lebih stable untuk daily usage.

Bagaimana Cara Upgrade ke Plasma 6.6

Proses upgrade akan berbeda-beda tergantung Linux distribution yang kamu gunakan. Berikut beberapa contoh:

Ubuntu/Debian-based

Untuk Ubuntu users, kamu bisa use KDE's official PPA. Open terminal dan jalankan:

sudo apt install software-properties-common
sudo add-apt-repository ppa:kubuntu-ppa/updates
sudo apt update && sudo apt upgrade

Fedora

Fedora users yang menggunakan KDE Plasma akan automatically mendapat update melalui official repositories:

sudo dnf upgrade

Arch Linux

Arch typically paling cepat untuk mendapat latest software:

sudo pacman -Syu

Important: Selalu backup data penting sebelum major updates. Meskipun KDE testing extensive, kamu tidak pernah tahu apa yang bisa happen di personal system kamu.
