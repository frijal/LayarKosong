Ubuntu vs Windows: Pengalaman Instal Pertama

Ini adalah rangkuman pengalaman saat pertama kali menginstal kedua sistem operasi di era 2010-an. Tujuannya sederhana: membandingkan apa saja yang "langsung bisa jalan" setelah instalasi selesai. Anggap ini sebagai catatan sejarah yang jujur dan santai.

Perbandingan Awal

Saat itu, baik Ubuntu maupun Windows 7 datang dalam CD berukuran sekitar 700 MB. Namun, perbedaan signifikan terasa pada kelengkapan software bawaan dan dukungan perangkat keras yang langsung aktif tanpa perlu mencari driver tambahan.

  • Software Bawaan: Ubuntu unggul dengan menyertakan paket office lengkap (OpenOffice), penampil PDF, dan klien torrent. Windows hanya menyediakan aplikasi dasar seperti WordPad dan Media Player.
  • Dukungan Hardware: Pengalaman saya menunjukkan Ubuntu langsung mengenali sebagian besar perangkat keras seperti kartu jaringan, Wi-Fi, dan printer. Di sisi lain, Windows seringkali memerlukan instalasi driver manual dari CD atau situs web produsen.
