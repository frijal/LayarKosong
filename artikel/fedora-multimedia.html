Panduan Lengkap Memutar Berkas Multimedia di Fedora Linux

Fakhrul Rijal 27 Mei 2026 Referensi: Dokumentasi Resmi Fedora & RPM Fusion

Secara default, Fedora tidak menyertakan beberapa codec multimedia karena alasan lisensi dan komitmen ketat mereka terhadap perangkat lunak open-source murni. Buat kamu pengguna baru yang baru saja menginstal OS ini (mungkin saat sedang migrasi dari Windows atau distro lain), wajar kalau tiba-tiba kebingungan kapan dan mengapa file MP4 atau MP3 kamu mendadak tidak bisa berputar di aplikasi bawaan.

Namun, jangan panik! Sistem kamu tidak rusak kok. Anda dapat dengan mudah menambahkan dukungan untuk berbagai format audio dan video dengan beberapa langkah sederhana melalui terminal. Mari kita bahas bagaimana cara mengatasi limitasi ini dengan aman.

Langkah 1: Aktifkan Repositori RPM Fusion 🚀

Paket-paket multimedia yang kita butuhkan (seperti codec H.264 untuk video atau AAC untuk audio) sayangnya tidak tersedia di repositori resmi Fedora. Oleh karena itu, langkah pertama yang mutlak harus dilakukan adalah menambahkan repositori pihak ketiga yang sangat dipercaya oleh komunitas, yaitu RPM Fusion.

Jalankan perintah berikut di aplikasi Terminal Anda untuk mengunduh dan memasang rilis RPM Fusion untuk paket free maupun non-free:

sudo dnf install https://download1.rpmfusion.org/free/fedora/rpmfusion-free-release-$(rpm -E %fedora).noarch.rpm https://download1.rpmfusion.org/nonfree/fedora/rpmfusion-nonfree-release-$(rpm -E %fedora).noarch.rpm

Setelah menekan Enter dan memasukkan password root, tunggu beberapa saat hingga proses download repositori selesai.

Langkah 2: Instal Codec Multimedia 📦

Setelah repositori RPM Fusion dipastikan aktif, Anda dapat menginstal grup paket multimedia dan sound-and-video untuk mendapatkan semua codec esensial secara serentak. Pendekatan "sapu jagat" ini adalah yang paling disarankan agar tidak ada ekstensi file yang terlewat.

sudo dnf groupupdate multimedia --setop="install_weak_deps=False" --exclude=PackageKit-gstreamer-plugin
sudo dnf groupupdate sound-and-video

Alternatif: Instalasi Secara Individual

Jika Anda tipikal pengguna Linux yang lebih suka meracik sistem seminimalis mungkin dan ingin menginstal paket secara individual tanpa bloatware, Anda dapat menggunakan deretan perintah berikut:

sudo dnf install gstreamer1-plugins-{bad-*,good-*,base} gstreamer1-plugin-openh264 gstreamer1-libav --exclude=gstreamer1-plugins-bad-free-devel
sudo dnf install lame* --exclude=lame-devel
sudo dnf group upgrade --with-optional Multimedia
Catatan Penting: gstreamer1-libav adalah nyawa utama di balik kemampuan sistem memutar format populer seperti MP4 yang menggunakan enkripsi standar industri.

Kesimpulan

Setelah menyelesaikan langkah-langkah di atas dan me-restart aplikasi pemutar media Anda (atau me-reboot sistem secara keseluruhan untuk memastikan daemon termuat ulang), Fedora Linux Anda kini siap beroperasi untuk memutar berbagai format audio dan video populer tanpa hambatan apa pun. Selamat menikmati sajian multimedia Anda!
