Salah satu perubahan signifikan di SparkyLinux 8.2 adalah pembaruan kernel. Rilis ini menggunakan kernel Linux 6.12.69 LTS sebagai default. Kernel ini akan mendapatkan dukungan jangka panjang hingga Desember 2026, menjamin keamanan dan stabilitas untuk penggunaan sehari-hari.

Tapi SparkyLinux terkenal dengan fleksibilitasnya. Jika kamu punya perangkat keras yang sangat baru (misalnya prosesor Intel Core Ultra seri 2 atau GPU NVIDIA RTX 50 series), kamu bisa menginstal kernel Linux 6.19.1 yang tersedia di repositori Sparky. Kernel 6.19 membawa banyak peningkatan driver dan dukungan untuk hardware generasi terbaru.

Selain itu, tersedia juga opsi kernel LTS alternatif: Linux 6.6.125 yang akan didukung hingga Desember 2026? (sebenarnya 6.6 adalah LTS jangka panjang hingga 2026). Jadi pengguna punya pilihan sesuai kebutuhan: stabilitas jangka panjang atau dukungan hardware mutakhir.

Bagi yang ingin mencoba kernel 6.19, cukup jalankan:

sudo apt update
sudo apt install linux-image-6.19.1-sparky linux-headers-6.19.1-sparky

Kemudian reboot dan pilih kernel yang diinginkan dari GRUB.

Bagi kalian yang sudah menjalankan SparkyLinux 8 (versi awal atau 8.1), tidak perlu repot-repot mengunduh ISO baru. Cukup jalankan perintah berikut di terminal:

Pengguna baru bisa langsung mengunduh ISO dari situs resmi dan membuat live USB menggunakan tool seperti Rufus (Windows) atau dd (Linux).
