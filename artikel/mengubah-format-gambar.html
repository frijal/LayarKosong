Mengubah Format Gambar Massal dengan ImageMagick

Jika Anda memiliki ratusan gambar yang harus diunggah ke website, ukurannya sering kali terlalu besar. Dengan ImageMagick, kita bisa mengubah format gambar secara massal agar lebih ringan.

Contoh Perintah

mogrify -format jpg *.png

Penjelasan

  • mogrify → perintah utama agar ImageMagick bekerja.
  • -format → menentukan format tujuan.
  • jpg → contoh format hasil (bisa diganti png, webp, dll).
  • *.png → semua file dengan format PNG di direktori aktif.
