Instalasi di Ubuntu

Cara pemasangannya gampang banget. Cukup buka terminal:

sudo apt install cycle

Instalasi di Fedora

Nah, untuk kamu pengguna Fedora, cara pemasangannya gampang banget. Cukup buka terminal dan jalankan perintah:

sudo yum install cycle

(Catatan: Jika Anda menggunakan versi Fedora yang lebih baru, perintahnya kemungkinan besar adalah `sudo dnf install cycle`).
