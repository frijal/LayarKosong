Ubuntu 25.10 Menghapus Startup Applications: Cara Autostart Sekarang

Ingin aplikasi favoritmu otomatis berjalan saat login ke Ubuntu? Dulu ada alat *Startup Applications*, tapi di Ubuntu 25.10, alat itu sudah tidak ada lagi. Mari kita bedah perubahannya.

Ubuntu 25.10 hadir dengan GNOME 49, yang membawa banyak perubahan dalam cara sesi ditangani, termasuk integrasi lebih dalam dengan **systemd**. Akibatnya, antarmuka grafis `gnome-startup-applications` yang dulu biasa kita gunakan kini telah dihapus.

Namun, hilangnya antarmuka ini **tidak berarti** kemampuan autostart aplikasi ikut hilang. Masih ada dua cara utama untuk mengaturnya.

Cara Manual: Membuat File `.desktop`

Untuk fleksibilitas penuh, seperti menjalankan skrip atau menambahkan argumen, Anda harus membuat file `.desktop` di direktori `~/.config/autostart/`. Berikut contohnya:

[Desktop Entry]
Type=Application
Version=1.0
Name=Jalankan Skrip Login
Comment=Menjalankan skrip kustom saat masuk
Exec=/home/<user>/scripts/startup.sh --verbose
StartupNotify=false
Terminal=false
X-GNOME-Autostart-enabled=true

**Tip:** Pastikan skrip Anda memiliki izin eksekusi (`chmod +x nama_skrip.sh`) dan gunakan path absolut untuk baris `Exec`.
