Atur mirror tercepat untuk Slackware (slackpkg)

1. Slackware (slackpkg)

Slackware menggunakan manajer paket `slackpkg`. Untuk memilih mirror, kita harus mengedit file `/etc/slackpkg/mirrors` dan memilih mirror terdekat.


# Buka daftar mirror Slackware
sudo nano /etc/slackpkg/mirrors
# Contoh: aktifkan mirror Indonesia (hapus tanda # di depan)
# ftp://kartolo.sby.datautama.net.id/slackware/slackware64-15.0/
# Simpan lalu update database slackpkg
sudo slackpkg update gpg
sudo slackpkg update
üîë Tips: Pilih mirror yang **resmi** dari Slackware atau komunitas terpercaya agar tetap aman.

2. NixOS (nix.conf mirror)

NixOS menggunakan sistem paket `nix`. Mirror bisa diatur dengan mengedit `/etc/nix/nix.conf` atau melalui `configuration.nix`.


# Edit file konfigurasi nix.conf
sudo nano /etc/nix/nix.conf
# Tambahkan mirror substituter (contoh: cache.nixos.org dan mirror lokal)
substituters = https://cache.nixos.org https://mirror.sjtu.edu.cn/nix-channels/store
trusted-public-keys = cache.nixos.org-1:xxxxxxxxxxxxxx mirror.sjtu.edu.cn-1:yyyyyyyyyyyyyy
# Simpan lalu jalankan
sudo nix-channel --update
nix-env -u
üåç Untuk daftar mirror resmi NixOS, kunjungi nixos.org/mirrors.

Kesimpulan

  • **Slackware** ‚Üí edit `/etc/slackpkg/mirrors` lalu update dengan `slackpkg`.
  • **NixOS** ‚Üí edit `/etc/nix/nix.conf` untuk menambahkan mirror substituter.

Dengan mirror yang tepat, baik Slackware maupun NixOS bisa melakukan update dan instalasi paket lebih cepat üöÄ.
