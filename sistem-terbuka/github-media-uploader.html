📖 Panduan Penggunaan & Keamanan

Halaman ini adalah alat bantu (utility) untuk mengupload file media (gambar/video) langsung ke repository GitHub dan langsung mendapatkan tautan **raw.githubusercontent.com** yang bisa disematkan ke mana saja.

Apa itu GitHub Token?

GitHub Token (Personal Access Token) bertindak sebagai "kunci digital" pengganti password. Token ini memberikan izin kepada aplikasi (halaman ini) untuk melakukan aksi spesifik—seperti menulis file—ke akun GitHub kamu secara aman.

Cara Mendapatkan Token (Fine-grained):

  1. Masuk ke akun GitHub kamu, lalu buka **Settings** > **Developer Settings**.
  2. Pilih **Personal access tokens** > **Fine-grained tokens**.
  3. Klik **Generate new token**.
  4. Isi nama token (misal: "Uploader Layar Kosong") dan pilih masa berlaku (Expiration).
  5. Pada bagian **Repository access**, pilih *"Only select repositories"* dan pilih repo tempat nyimpen media kamu.
  6. Pada **Permissions**, cari **Contents** dan ubah aksesnya menjadi **"Read and write"**.
  7. Klik **Generate token** dan salin kodenya. *(Ingat: Kode ini hanya muncul sekali!)*
⚠️ Catatan Keamanan:

Token disimpan secara lokal di browser kamu jika kamu menyetujui fitur "Save Password". Jangan pernah memberikan kode token kamu kepada siapa pun. Token ini memberikan akses penuh untuk memodifikasi file di repository yang kamu pilih.
