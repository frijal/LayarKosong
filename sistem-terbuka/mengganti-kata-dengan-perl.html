Mengganti Kata dalam Banyak Dokumen dengan Perl

Terkadang kita perlu merevisi dokumen dengan mengganti kata tertentu di banyak file sekaligus. Jika dilakukan manual, tentu sangat melelahkan. Untungnya, Perl menyediakan solusi cepat dengan satu baris perintah.

Perintah Dasar

Contoh kasus: Anda ingin mengganti kata "Desember" menjadi "Januari" di semua file laporan bulanan yang berformat .txt. Dengan Perl, cukup gunakan perintah berikut di terminal:

perl -pi -w -e 's/Desember/Januari/g;' *.txt

Perintah ini akan mencari dan mengganti semua kemunculan kata "Desember" di setiap file .txt di direktori saat ini.

Mengganti URL atau Teks dengan Karakter Khusus

Jika teks yang ingin Anda ganti mengandung karakter / (seperti pada URL), Anda bisa menggunakan pemisah lain, misalnya #, |, atau :.

perl -pi -w -e 's#//dalam.web.id/home#//dalam.web.id/sitemap#g' *.html

Keunggulan Menggunakan Perl

  • Cepat: Satu perintah singkat untuk mengganti semua kata di banyak file.
  • Efisien: Tidak perlu membuka file satu per satu secara manual.
  • Powerful: Dukungan Regular Expression (regex) memungkinkan pencarian dan penggantian yang sangat kompleks.

Perlu diingat, metode ini paling cocok untuk file teks biasa (plain text), seperti .txt, .html, .css, .xml, dan file kode sumber lainnya.
