Install Sabily (Ubuntu Muslim Edition)

Artikel oleh: Rijal Fakhrul

Sabily adalah distro Linux berbasis Ubuntu yang dibuat khusus untuk pengguna Muslim. Distro ini hadir dengan koleksi software Islami seperti jadwal sholat, pengingat adzan, Al-Qur’an digital, hingga aplikasi belajar bahasa Arab. Bisa dibilang, Sabily adalah "Ubuntu rasa Islami" yang tampil modern tapi tetap ringan.

Artikel ini akan membahas bagaimana cara instalasi Sabily dari awal, lengkap dengan tips kecil biar prosesnya lancar. Cocok buat kamu yang ingin belajar Linux dengan nuansa religi, atau sekadar nostalgia dengan distro unik yang satu ini.

Persiapan Sebelum Instalasi

Sebelum mulai instalasi, pastikan kamu sudah menyiapkan beberapa hal penting berikut:

  • File ISO Sabily (biasanya berbasis Ubuntu 10.04 atau 11.04).
  • USB bootable atau DVD untuk instalasi.
  • Koneksi internet (opsional, untuk update dan repo tambahan).
  • Backup data penting di komputer kamu.

Jika semuanya sudah siap, kamu bisa langsung mengikuti panduan berikut yang dibuat oleh Rijal Fakhrul dan dipublikasikan di Scribd. Panduan ini menjelaskan langkah-langkah instalasi Sabily dari awal hingga selesai, termasuk pengaturan partisi dan user.

Tentang Sabily

Sabily dulu dikenal juga sebagai Ubuntu Muslim Edition. Tujuannya sederhana: memudahkan umat Muslim untuk menggunakan sistem operasi open source dengan aplikasi yang relevan dengan kebutuhan spiritual sehari-hari.

Di dalam Sabily, kamu bisa menemukan aplikasi seperti:

  • Minbar – pengingat waktu sholat dan adzan otomatis.
  • Zekr – aplikasi Al-Qur’an digital dengan tafsir dan terjemahan.
  • Hijra Applet – penanggalan Hijriyah di desktop.
  • Thawab – pustaka Islam elektronik.

Penutup

Instalasi Sabily tidak jauh berbeda dengan Ubuntu, hanya saja tampilannya lebih Islami dan dilengkapi software religi. Walaupun proyek ini sudah tidak aktif sejak beberapa tahun lalu, Sabily tetap meninggalkan kesan kuat di komunitas pengguna Linux Muslim.

Kalau kamu ingin mencoba distro modern dengan semangat serupa, kamu bisa eksplor proyek-proyek turunan Ubuntu terbaru, lalu menambahkan aplikasi Islami secara manual.
