Apa Itu Struktur Direktori Linux?

Struktur berkas (filesystem hierarchy) di Linux mirip seperti pohon, di mana akar utamanya adalah / (root). Semua folder lain seperti /home atau /usr adalah cabangnya. Nggak ada drive huruf seperti di Windows (C:, D:, E:), semua berkas dan perangkat digabung di bawah satu akar: /.

Filosofinya sederhana: â€œsemua adalah berkas.â€ Bahkan perangkat keras seperti keyboard, mouse, dan hard disk juga direpresentasikan sebagai file di dalam sistem.

Penjelasan Folder-Folder Utama di Linux

  • / â†’ Akar dari seluruh sistem. Semua hal dimulai dari sini.
  • /bin â†’ Berisi perintah dasar seperti ls, cp, mv, dan lainnya. Tanpa ini, sistem nggak bisa berfungsi.
  • /boot â†’ Di sinilah kernel Linux dan berkas bootloader seperti GRUB disimpan.
  • /dev â†’ Tempat semua perangkat keras direpresentasikan sebagai berkas. Misalnya /dev/sda untuk hard disk.
  • /etc â†’ Berisi file konfigurasi sistem. Kalau kamu suka ngoprek, pasti sering main ke sini.
  • /home â†’ Tempat penyimpanan data pribadi pengguna. Misalnya /home/rijal.
  • /lib â†’ Menyimpan pustaka (library) penting yang dibutuhkan oleh sistem dan program di /bin.
  • /media dan /mnt â†’ Biasanya digunakan untuk mengaitkan (mount) perangkat eksternal seperti flashdisk.
  • /opt â†’ Berisi aplikasi tambahan yang tidak berasal dari repositori utama Linux.
  • /root â†’ Folder khusus untuk akun administrator (root user).
  • /tmp â†’ Folder sementara untuk file yang tidak perlu disimpan lama.
  • /usr â†’ Salah satu direktori terbesar. Menyimpan aplikasi, pustaka, dan dokumentasi sistem.
  • /var â†’ Berisi file yang sering berubah, seperti log sistem, cache, dan email.

Dengan mengenal fungsi tiap folder, kamu jadi lebih nyaman menjelajahi sistem Linux. Misalnya, kalau mau ubah pengaturan jaringan, kamu bakal tahu harus buka /etc/network.

Kenapa Penting Paham Struktur Ini?

Karena paham struktur direktori artinya kamu paham â€œanatomiâ€ Linux. Ini penting kalau kamu ingin jadi pengguna mahir, sysadmin, atau devops engineer. Banyak masalah Linux yang bisa diselesaikan hanya dengan tahu folder mana yang harus dikunjungi.

Bahkan, ketika kamu menginstal software dari terminal, sistem akan otomatis menaruh file-nya di lokasi yang sesuai: biner ke /usr/bin, pustaka ke /usr/lib, dan konfigurasi ke /etc. Semua tertata rapi dan konsisten.
