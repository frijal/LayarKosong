Membuat USB Bootable Ubuntu Tanpa USB Creator

Umumnya, Ubuntu menyediakan aplikasi USB Creator karena file ISO-nya tidak umumnya menyertakan fitur Hybrid ISO. Hasil dapat berbeda tergantung perangkat, versi paket, dan konfigurasi sistem. Namun, jika Anda menggunakan distro lain atau ingin cara yang lebih universal, perintah dd adalah solusinya.

Menggunakan Perintah dd

Jika file ISO Linux Anda sudah berjenis Hybrid ISO (seperti Fedora atau rilis Ubuntu modern), Anda bisa membuat USB bootable langsung dari terminal menggunakan perintah dd. Pastikan Anda sangat berhati-hati saat menentukan nama perangkat USB (misalnya /dev/sdb) agar tidak salah format hard disk utama.

# Ganti 'ubuntu.iso' dengan nama file Anda
# Ganti '/dev/sdX' dengan path USB Anda (cek dengan 'lsblk')
sudo dd if=ubuntu.iso of=/dev/sdX bs=4M status=progress oflag=sync
