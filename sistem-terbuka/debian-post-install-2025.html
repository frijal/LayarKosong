Konfigurasi yang Masih Biasa Dilakuin Orang

  • Tambah repo `contrib non-free non-free-firmware`
  • Install paket favorit: vim, git, curl, htop, synaptic, chrome/firefox-esr, dll.
  • Nyalain firewall (`ufw`), printing (`cups`), atau snapshot (`snapper`).
  • Yang suka bleeding-edge tambah backports.
  • Setup dotfiles, SSH keys, GPG, shell favorit (zsh, fish, dll).

Kesimpulan Akhir

Kalau kamu pengen Debian di Thinkpad X230 (atau laptop lain yang nggak terlalu baru):

  1. Download ISO resmi dengan non-free firmware
  2. Pilih desktop environment saat instal
  3. Reboot ‚Üí langsung bisa kerja/sekolah/ngoding/nonton YouTube
  4. Tinggal tambah paket-paket kesukaanmu, selesai!
