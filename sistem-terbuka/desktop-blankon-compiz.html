Eksperimen Desktop BlankOn: Compiz, Gnome-Do, dan XFishtank

Sebuah catatan santai tentang mengisi waktu luang dengan mengoprek desktop BlankOn. Eksperimen ini menggabungkan efek visual Compiz, peluncur aplikasi Gnome-Do, dan hiasan "akuarium" xfishtank.

Komponen Utama Setup

  • **Distro:** BlankOn (berbasis Debian)
  • **Efek Visual:** Compiz untuk animasi jendela, exposÃ©, dan transisi desktop yang halus.
  • **Peluncur Aplikasi:** Gnome-Do untuk akses cepat ke aplikasi dengan shortcut keyboard.
  • **Pemanis Visual:** `xfishtank` untuk menampilkan animasi ikan di desktop.
  • **Perekam Layar:** GTK-RecordMyDesktop untuk merekam aktivitas desktop ke format `.ogv`.

Contoh Perintah Instalasi

Perintah ini adalah contoh untuk sistem berbasis Debian/Ubuntu. Nama paket dapat bervariasi tergantung rilis distro Anda.

sudo apt update
sudo apt install compiz compiz-plugins-extra gnome-do xfishtank recordmydesktop

Setelah semua terinstal, Anda bisa mulai mengonfigurasi Compiz melalui CCSM (CompizConfig Settings Manager), mengatur tema Gnome-Do, dan menjalankan `xfishtank` dari terminal.
