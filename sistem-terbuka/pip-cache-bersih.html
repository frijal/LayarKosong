Folder pip cache dan http-v2 bisa membengkak seiring waktu. Yuk kita bersihkan dengan cara yang aman dan santai!

üîç 1. Cek ukuran dan isi cache pip

Sebelum membersihkan, kamu bisa lihat dulu seberapa besar cache pip di sistemmu:

pip cache info

Ini akan menampilkan lokasi dan ukuran folder cache, termasuk bagian http-v2 dan wheels.

üßπ 2. Bersihkan cache pip secara otomatis

Untuk membersihkan semua isi cache pip (termasuk http-v2 dan wheels), gunakan:

pip cache purge

Perintah ini akan menghapus semua file cache yang disimpan pip, termasuk file index dan paket yang pernah diunduh.

üßº 3. Hapus cache pip secara manual (opsional)

Kalau kamu ingin kontrol penuh atau pip versi lama tidak mendukung cache purge, kamu bisa hapus folder cache langsung:

rm -rf ~/.cache/pip

Atau jika kamu hanya ingin hapus bagian tertentu:

rm -rf ~/.cache/pip/http
rm -rf ~/.cache/pip/http-v2
rm -rf ~/.cache/pip/wheels

üß† 4. Cegah cache membesar di masa depan

Kalau kamu ingin pip tidak menyimpan cache saat menginstal paket, tambahkan opsi --no-cache-dir:

pip install nama-paket --no-cache-dir

‚ö†Ô∏è Catatan penting

  • Membersihkan cache tidak akan menghapus paket yang sudah terinstal, hanya file cache.

  • Tidak ada risiko merusak sistem Python kamu selama kamu hanya menghapus cache.
