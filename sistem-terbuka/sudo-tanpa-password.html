Setting Sudo Tanpa Password di Ubuntu

Hai, para pengguna Linux! Kalau kamu pakai komputer atau laptop sendirian dan udah yakin nggak bakal ada yang iseng, mungkin tips ini bisa bikin hidup sedikit lebih cepat.

Tentunya saat ini sebagian besar dari kita sudah jenuh untuk mengetik password ketika akan melakukan kegiatan *root*... Ya, kan? Memasang dan menghapus paket, koneksi ke sebuah jaringan yang membutuhkan akses *Key-Ring*, dan sebagainya. Kadang ngerasa ribet aja gitu, padahal cuma mau `sudo apt update`.

Jika yakin bahwa komputer/laptop memang kita gunakan sendiri, dan tak ada campur tangan orang lain yang akan iseng mengubah konfigurasinya, tentunya Ubuntu juga harus mempercayai kita (harus saling percaya, dong).

Untuk dapat langsung beraksi sebagai *root*, bisa menggunakan paket `fakeroot` atau untuk kegiatan yang menggunakan GUI, bisa dengan sedikit mengubah konfigurasi pada berkas `sudoers` yang berada di `/etc/sudoers`.

Langkah-Langkah "Oprek" File Sudoers

Nah, kita akan fokus ke cara kedua, yaitu mengedit file `/etc/sudoers`. Tapi ingat, jangan pernah edit file ini pakai Gedit atau editor teks biasa, ya! Selalu gunakan `visudo`.

**1. Buka Terminal Terdekat**

Ketikkan perintah sakti ini di terminal. Perintah `visudo` adalah cara aman untuk mengedit file `sudoers` karena dia akan mengecek sintaks-nya sebelum menyimpan. Jadi, kita nggak bakal "ke-kunci" di luar sistem kalau ada salah ketik.

sudo visudo

**2. Cari Baris Admin**

Setelah editor-nya terbuka (biasanya pakai Nano atau Vi/Vim, tergantung settingan default), langsung menuju bagian paling bawah. Kamu akan menemukan baris yang mirip seperti ini:

%admin ALL=(ALL) ALL

Atau bisa juga baris untuk grup `sudo` (di Ubuntu modern biasanya pakai grup `sudo`):

%sudo   ALL=(ALL:ALL) ALL

Cari saja baris yang sesuai dengan grup user kamu (biasanya antara `admin` atau `sudo`).

**3. Tambahkan Ajaib "NOPASSWD"**

Sekarang, tambahkan `NOPASSWD:` di akhir baris tersebut.

Jika baris aslimu `%admin ALL=(ALL) ALL`, ubah menjadi:

%admin ALL=(ALL) NOPASSWD: ALL

Atau, jika baris aslimu `%sudo ALL=(ALL:ALL) ALL`, ubah menjadi:

%sudo   ALL=(ALL:ALL) NOPASSWD: ALL

**4. Simpan dan Tutup**

Selesai! Sekarang simpan perubahan itu dan tutup editor `visudo`.

  • Jika pakai **Nano** (biasanya default): Tekan `Ctrl + X`, lalu `Y`, lalu `Enter`.
  • Jika pakai **Vim**: Tekan `Esc`, lalu ketik `:wq!`, lalu `Enter`.

Coba buka terminal baru dan ketikkan perintah `sudo apt update`. Voila! Seharusnya sekarang langsung jalan tanpa minta password lagi.

**⚠️ PERINGATAN KERAS!**
Setelah ini harap waspada jika komputer/laptop anda digunakan oleh orang lain yang diragukan kejujurannya... Ini serius. Siapapun yang bisa mengakses user kamu, dia bisa melakukan APAPUN di sistem kamu sebagai root tanpa halangan. **Gunakan risiko ini dengan bijak!**

Kesimpulan

Dengan settingan ini, pekerjaan *sysadmin* di mesin sendiri jadi lebih sat-set-sat-set. Tapi ingat, kenyamanan ini dibayar dengan sedikit penurunan keamanan. Pastikan kamu satu-satunya yang pakai akun tersebut dan selalu kunci komputermu (`Super + L`) kalau lagi ditinggal, ya!
