Perbaikan Manual GRUB Bootloader

Jika GRUB rusak atau hilang, Anda bisa memperbaikinya secara manual menggunakan Live USB. Cara ini memberi Anda kontrol penuh atas setiap langkah yang dilakukan pada sistem.

Langkah-Langkah Perbaikan

  1. Boot ke Live USB Debian/Ubuntu dan buka terminal.
  2. Cari partisi root Linux Anda dengan perintah:
    sudo fdisk -l
  3. Mount partisi tersebut (ganti /dev/sda1 sesuai partisi Anda):
    sudo mount /dev/sda1 /mnt
  4. Bind direktori sistem yang penting:
    for i in /dev /proc /sys; do sudo mount --bind $i /mnt$i; done
  5. Masuk ke lingkungan chroot:
    sudo chroot /mnt
  6. Instal ulang GRUB ke disk (bukan partisi):
    grub-install /dev/sda
update-grub
  7. Keluar, unmount, dan reboot:
    exit
sudo umount /mnt/{dev,proc,sys}
sudo umount /mnt
sudo reboot
