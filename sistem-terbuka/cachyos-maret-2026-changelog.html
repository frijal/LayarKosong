🚀 CachyOS Maret 2026: Installer Animasi, Winboat, dan Handheld Makin Gahar!

Halo Sobat Linux! 👋 Setelah sukses dengan rilis sebelumnya, CachyOS kembali lagi dengan pembaruan besar di bulan Maret 2026. Buat kamu yang belum tahu, CachyOS adalah distro Linux berbasis Arch yang difokuskan untuk performa tinggi, terutama buat gaming. Nah, rilis kali ini (kode internal 260308) hadir dengan segudang fitur baru dan perbaikan yang membuat pengalaman makin mantap. Yuk, kita bedah satu per satu!

📌 Apa Itu Rilis CachyOS Maret 2026?

Rilis ini adalah update kedua di tahun 2026. Isinya banyak perubahan mulai dari tampilan installer yang kini kasih pratinjau desktop animasi, integrasi Winboat buat virtualisasi Windows mulus, sampai edisi Handheld yang dirombak total. Ada juga kernel linux-cachyos yang sekarang pakai sistem rilis terpisah, dan banyak perbaikan bug yang meningkatkan stabilitas.

🧑💻 Siapa di Balik Rilis Ini?

Tim pengembang CachyOS yang dipimpin oleh para kontributor hebat, plus dukungan komunitas global. Mereka bekerja keras buat ngepush update ini, termasuk penambahan mirror baru dari Rusia (jura12, cachy-arch.ru), Swedia (Zyner), dan Kanada (All Things Linux). Jadi, user di berbagai belahan dunia bisa download lebih cepat.

📅 Kapan Rilis Tersedia?

Rilis resmi dirilis pada pertengahan Maret 2026. ISO sudah bisa diunduh sejak tanggal 8 Maret 2026 (lihat kode folder 260308). Buat pengguna existing, cukup update lewat package manager.

📍 Di Mana Bisa Mendapatkan?

ISO bisa di-download dari mirror resmi: CDN utama atau CDN2 untuk edisi Desktop, dan link terpisah untuk edisi Handheld. Semua link tersedia di artikel ini.

⚡ Kenapa Harus Update?

Karena banyak peningkatan yang bikin pengalaman makin oke: installer lebih informatif, dukungan hardware handheld makin matang (Steam Deck, Lenovo Legion Go), ukuran initramfs jadi lebih kecil berkat chwd, dan pastinya performa gaming makin juara. Juga ada fitur Winboat yang memudahkan virtualisasi Windows tanpa rumit.

🛠️ Bagaimana Cara Update atau Instal?

Buat pengguna lama, cukup jalankan perintah berikut di terminal:

sudo pacman -Syu

Buat yang baru mau nyobain, tinggal download ISO terbaru, burning ke USB, dan install. Installer sekarang juga kasih preview desktop sebelum kamu pilih, jadi kamu bisa liat dulu tampilan Plasma, GNOME, Niri, atau COSMIC sebelum install.

📋 Rincian Perubahan (Changelog) — Lengkap!

🎨 Installer Makin Canggih

  • Pratinjau Desktop Animasi: Pas milih desktop environment (Plasma, GNOME, Niri, COSMIC), sekarang muncul preview dalam format GIF/WebP. Jadi kamu bisa liat langsung kayak apa tampilannya sebelum install.
  • Dukungan JPEG XL: Buat ngurangin ukuran gambar di pilihan desktop.
  • Cachy-Update Diaktifkan Default: Khusus instalasi GNOME dan KDE, tools update otomatis langsung nyala.
  • Deteksi Microcode Otomatis: Installer sekarang cuma pasang microcode sesuai hardware (AMD atau Intel), tidak kedua-duanya.
  • Peringatan Partisi EFI Kecil: Pesan error lebih jelas kalau partisi EFI terlalu kecil.
  • Daftar DE Diurutkan: Dari yang mudah (seperti Plasma) sampai advanced (window manager).

🖥️ CachyOS-Welcome

  • Tombol Winboat: Bisa langsung install dan enable Winboat buat virtualisasi Windows pake Docker dengan mulus.
  • DNS FFMUC: Opsi DNS server baru.
  • Terjemahan Ukraina: Sekarang makin banyak bahasa.
  • Navigasi Keyboard: Aksesibilitas meningkat.

⚙️ chwd & Kernel

  • Initramfs Ngecil: Untuk konfigurasi NVIDIA dGPU, ukuran initramfs berkurang drastis.
  • Linux-cachyos Rilis Terpisah: Sekarang tiap versi kernel punya repo sendiri, tidak lagi pake patch tunggal (0001-cachyos-base-all.patch).

🌐 Sistem & Tools

  • cachyos-rate-mirrors: Pengalaman di China dan Rusia membaik dengan pengecekan mirror sebelum dirating.
  • Domain Regulasi Wireless: Otomatis set berdasarkan timezone.
  • Bugreport Redacted: IP, username, hostname, MAC address disembunyikan di cachyos-bugreport.sh.
  • Website Rework: Desain website lebih modern.
  • Issue Templates di GitHub: Biar laporan bug lebih rapi.
  • Mirror Baru: Rusia (jura12, cachy-arch.ru), Swedia (Zyner), Kanada (All Things Linux).

🔧 Fixes (Perbaikan)

  • bcachefs Dihapus: Karena butuh bcachefs-dkms, sementara dihapus dari opsi filesystem.
  • Enkripsi LUKS2: Diperbaiki untuk device tertentu.
  • Display Manager "ly": Kini bisa diaktifkan.
  • Handheld Profiles: chwd sekarang support lebih baik untuk GPU handheld, dan fwupd aktif untuk Lenovo.

🎮 Edisi Handheld (Revolusi!)

  • gamescope-session-cachyos: Gantikan gamescope-session-plus, fork dari Valve, mendukung firmware update Steam Deck dan Lenovo Legion Go.
  • plasma-login-manager: Gantikan SDDM sebagai login manager.
  • Limine Default: Bootloader baru dengan snapshot otomatis (systemd-boot tetap bisa dipilih).
  • Installer Handheld Digabung: Sekarang jadi satu dengan edisi desktop di Calamares.
  • ISO Wayland: Sekarang defaultnya pake Wayland, bukan X11.
