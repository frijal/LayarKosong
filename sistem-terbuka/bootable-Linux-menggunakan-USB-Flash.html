Membuat Bootable Linux dengan USB

Membuat USB bootable adalah langkah pertama untuk menginstal atau mencoba sistem operasi Linux. Prosesnya sederhana dan hanya memerlukan beberapa langkah.

Langkah-langkah Membuat USB Bootable

  1. Unduh File ISO Linux: Pilih dan unduh file ISO dari situs resmi distro Linux yang Anda inginkan (misalnya Ubuntu, Mint, Fedora).
  2. Siapkan USB Flash Drive: Gunakan USB flash drive dengan kapasitas minimal 4 GB. Pastikan Anda sudah mem-backup semua data penting di dalamnya, karena proses ini akan menghapus seluruh isinya.
  3. Gunakan Software Pembuat Bootable: Gunakan aplikasi seperti Rufus (untuk Windows) atau Balerina Etcher (untuk Windows, macOS, Linux) untuk "membakar" file ISO ke USB.
  4. Boot dari USB: Restart komputer Anda, masuk ke menu BIOS/UEFI (biasanya dengan menekan F2, F12, atau Del saat startup), dan atur urutan boot agar USB drive menjadi prioritas utama.
