Skrip fedora-mirror.sh: Aktifkan Mirror Tercepat

Fedora menggunakan `dnf` sebagai manajer paket. Untuk mempercepat update dan instalasi, kita bisa mengaktifkan plugin **fastestmirror** yang akan otomatis memilih server mirror tercepat. Skrip berikut akan melakukannya untuk Anda.

⚠️ Jalankan dengan `sudo` karena skrip ini akan mengubah file konfigurasi sistem di `/etc/dnf/dnf.conf`.

Kode Skrip

#!/bin/bash
# fedora-mirror.sh - Mengaktifkan mirror DNF tercepat di Fedora
set -e
echo "[INFO] Mem-backup konfigurasi DNF..."
sudo cp /etc/dnf/dnf.conf /etc/dnf/dnf.conf.bak
echo "[INFO] Mengaktifkan 'fastestmirror' di /etc/dnf/dnf.conf..."
if ! grep -q "fastestmirror=True" /etc/dnf/dnf.conf; then
    echo "fastestmirror=True" | sudo tee -a /etc/dnf/dnf.conf
fi
echo "[INFO] Membersihkan cache DNF..."
sudo dnf clean all
echo "[SELESAI] Plugin 'fastestmirror' berhasil diaktifkan!"
