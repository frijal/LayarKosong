Fix WiFi Realtek di Linux (Sinyal Lemah)

Jika Anda menggunakan adapter WiFi Realtek di Linux (seperti pada beberapa model laptop) dan mengalami sinyal yang lemah atau tidak stabil, masalah ini seringkali bisa diatasi dengan mengatur ulang modul driver dan memilih antena yang tepat.

Langkah-langkah Perbaikan

Berikut adalah langkah-langkah yang bisa Anda coba melalui terminal. Perintah ini secara spesifik menargetkan driver rtl8723be, salah satu yang paling umum bermasalah.

# Cek nama interface WiFi Anda (misal: wlan0, wlp2s0)
iwconfig
# Matikan interface WiFi
sudo ifconfig wlan0 down
# Bongkar pasang modul driver dengan opsi antena
sudo modprobe -r rtl8723be
sudo modprobe rtl8723be ant_sel=2
# Aktifkan kembali interface WiFi
sudo ifconfig wlan0 up

Catatan: Ganti wlan0 dengan nama interface WiFi Anda. Opsi ant_sel=2 memaksa driver untuk menggunakan antena kedua, yang seringkali memiliki penerimaan sinyal lebih baik. Jika tidak berhasil, Anda bisa mencoba ant_sel=1.
