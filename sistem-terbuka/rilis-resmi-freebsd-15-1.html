FreeBSD 15.1-RELEASE Resmi Dirilis dengan Dukungan Linux 7.0

Kabar gembira bagi para pengguna dan penggemar sistem operasi legendaris ini. Tepat pada 16 Juni 2026, Tim Rekayasa Rilis FreeBSD secara resmi mengumumkan ketersediaan FreeBSD 15.1-RELEASE. Ini merupakan rilis kedua dari cabang stable/15, kelanjutan dari seri 15 yang pertama kali diperkenalkan tahun lalu. Buat kamu yang belum tahu, FreeBSD adalah sistem operasi turunan Unix yang terkenal tangguh, aman, dan banyak dipakai di server, embedded system, hingga infrastruktur cloud raksasa.

Rilis kali ini bukan sekadar pembaruan minor. Ada banyak peningkatan signifikan yang membuat FreeBSD semakin relevan di era komputasi modern. Mulai dari driver nirkabel berbasis Linux 7.0, dukungan penuh standar bahasa pemrograman C23, hingga pembaruan Unicode 17.0.0 dengan ribuan karakter baru. Semua itu dikemas dalam satu paket yang stabil dan siap produksi.

🆕 Sorotan Fitur FreeBSD 15.1

Pembaruan paling mencolok ada di sisi jaringan. Driver iwlwifi(4) dan driver berbasis LinuxKPI lainnya kini telah diperbarui ke basis Linux v7.0. Artinya, dukungan untuk chipset nirkabel modern—terutama Intel—jauh lebih baik, lebih stabil, dan lebih cepat. Bagi pengguna laptop atau perangkat yang mengandalkan WiFi, ini langkah besar.

Selain itu, image cloud FreeBSD kini menggunakan sistem basis terpaket. Artinya, pkg(8) sudah terpasang secara bawaan di image tersebut, dan pembaruan paket sistem basis bisa berjalan otomatis pada booting pertama. Ini mempermudah administrator sistem yang mengelola banyak instans di cloud.

Fitur lain yang menarik adalah variabel kern.sched.name yang bisa diatur saat boot untuk memilih penjadwal kernel. Jadi kamu bisa menentukan sendiri algoritma penjadwalan yang paling cocok dengan beban kerja, tanpa perlu kompilasi ulang kernel.

Di ranah bahasa pemrograman, kemajuan signifikan telah dicapai menuju dukungan penuh standar C23. FreeBSD umumnya menjadi platform pilihan para pengembang C, dan kini lebih siap untuk kode modern. Hasil dapat berbeda tergantung perangkat, versi paket, dan konfigurasi sistem. Lalu, dukungan Unicode diperbarui ke Unicode 17.0.0 dan CLDR 48, menambahkan 4.803 karakter baru termasuk simbol, emoji, dan aksara dari berbagai bahasa. Multilingual support makin oke.

📚 Dedikasi untuk Peter G. Neumann

Proyek FreeBSD mendedikasikan rilis 15.1 ini untuk mengenang Peter G. Neumann, seorang pionir keamanan sistem operasi. Beliau lama berkolaborasi dengan pengembang FreeBSD dalam riset keamanan berbasis kapabilitas. Melalui kontribusinya di proyek legendaris seperti Multics, Neumann meletakkan banyak konsep dasar keamanan OS yang kita pakai sampai sekarang. Rilis ini menjadi penghormatan kecil atas warisannya.

🖥️ Ketersediaan Arsitektur

FreeBSD 15.1-RELEASE tersedia untuk enam arsitektur: amd64 (x86_64), aarch64 (ARM64), armv7, powerpc64, powerpc64le, dan riscv64. Pemasangan bisa dilakukan dari image ISO bootable, USB memory stick, atau melalui jaringan. Image kartu SD juga disediakan untuk perangkat ARM seperti Raspberry Pi.

📦 Ragam Image dan Cara Penggunaannya

Bagi yang belum familiar, FreeBSD menyediakan beberapa jenis image untuk kebutuhan berbeda. Berikut penjelasan ringkas masing-masing, lengkap dengan perintah dd untuk menulisnya ke USB drive. Hati-hati memastikan target perangkat (of=/dev/da0) sudah benar, jangan sampai salah pilih partisi.

dvd1

Image paling lengkap. Berisi sistem dasar, dokumentasi, set debugging, dan paket pra-bangun untuk workstation grafis. Bisa dipakai boot ke mode penyelamatan livefs. Cocok untuk instalasi penuh via DVD atau USB.

# dd if=FreeBSD-15.1-RELEASE-amd64-dvd1.iso \
 of=/dev/da0 bs=1m conv=sync

disc1

Hanya sistem dasar tanpa paket pihak ketiga. Tetap mendukung mode livefs. Cocok buat instalasi bersih.

# dd if=FreeBSD-15.1-RELEASE-amd64-disc1.iso \
 of=/dev/da0 bs=1m conv=sync

bootonly

Image kecil untuk boot via CD/USB, tapi instalasi dilakukan melalui jaringan (HTTP/FTP). Tidak ada paket bawaan.

# dd if=FreeBSD-15.1-RELEASE-amd64-bootonly.iso \
 of=/dev/da0 bs=1m conv=sync

memstick

Image untuk USB memory stick. Bisa dipakai langsung tanpa perlu burning CD. Tidak ada paket pihak ketiga.

# dd if=FreeBSD-15.1-RELEASE-amd64-memstick.img \
 of=/dev/da0 bs=1m conv=sync

mini-memstick

Mirip bootonly, untuk USB kecil, tanpa set instalasi. Instalasi tetap butuh jaringan.

# dd if=FreeBSD-15.1-RELEASE-amd64-mini-memstick.img \
 of=/dev/da0 bs=1m conv=sync

Image Kartu SD ARM

Untuk perangkat ARM (misal Raspberry Pi). Sudah berisi instalasi penuh, bisa langsung ditulis ke SD card minimal 5 GB. Pengguna awal: freebsd / sandi freebsd dan root root (segera ganti setelah boot).

# dd if=FreeBSD-15.1-RELEASE-arm64-aarch64-RPI.img \
 of=/dev/da0 bs=1m conv=sync

☁️ Image Mesin Virtual & Platform Cloud

FreeBSD 15.1 juga menyediakan image pra-instal untuk mesin virtual dalam format QCOW2, VHD, VMDK, dan raw bagi arsitektur amd64, i386, AArch64, dan riscv64. Selain itu, tersedia langsung di tiga raksasa cloud publik:

Amazon® EC2™

AMI ID bisa diambil lewat Systems Manager Parameter Store. Tersedia varian UFS dan ZFS, dengan atau tanpa cloud-init, untuk arsitektur amd64 dan arm64. Marketplace:

  • AMI UFS amd64
  • AMI ZFS amd64
  • AMI UFS arm64
  • AMI ZFS arm64

Google® Compute Engine™

Gunakan perintah gcloud compute instances create dengan image rilis-resmi-freebsd-15-1-amd64-ufs dari proyek freebsd-org-cloud-dev. Tersedia juga di GCP Marketplace.

% gcloud compute instances create INSTANCE \
 --image rilis-resmi-freebsd-15-1-amd64-ufs \
 --image-project=freebsd-org-cloud-dev
% gcloud compute ssh INSTANCE

Microsoft® Azure™

Image virtual mesin diharapkan segera tersedia di Azure Marketplace.

🔗 Unduhan Langsung dan Instalasi

Semua image bisa diunduh dari server resmi FreeBSD:

  • ISO: https://download.freebsd.org/releases/ISO-IMAGES/15.1/
  • VM: https://download.freebsd.org/releases/VM-IMAGES/15.1-RELEASE/
  • BASIC-CI: https://download.freebsd.org/releases/CI-IMAGES/15.1-RELEASE/
  • OCI container: https://download.freebsd.org/releases/OCI-IMAGES/15.1-RELEASE/ (juga di Docker Hub)

Untuk panduan instalasi atau upgrade dari versi sebelumnya, kunjungi tautan resmi ini.

🛡️ Dukungan Jangka Panjang

FreeBSD 15.1-RELEASE akan didukung hingga 31 Maret 2027. Sementara itu, FreeBSD 15.0-RELEASE mencapai akhir masa pakai (EOL) pada 30 September 2026. Seluruh seri rilis FreeBSD 15 dijadwalkan mendapat dukungan hingga 31 Desember 2029. Jadi, ini saat yang tepat untuk mulai migrasi atau upgrade.

Seperti biasa, rilis ini membuktikan bahwa FreeBSD tetap menjadi pilihan solid bagi mereka yang menginginkan sistem operasi yang stabil, transparan, dan penuh kendali. Dengan peningkatan signifikan di sisi driver, toolchain, dan dukungan cloud, FreeBSD 15.1 siap menghadapi lanskap teknologi yang terus berubah.
