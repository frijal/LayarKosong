Pembaruan Masif: Rilis Resmi FreeBSD 15.1

**Diumumkan oleh Tim Rekayasa Rilis FreeBSD pada 16 Juni 2026**

Bagi kamu para administrator sistem, *developer*, maupun pegiat teknologi yang mendambakan sistem operasi yang super stabil, hari ini ada kabar gembira. Tim Rekayasa Rilis FreeBSD akhirnya mengumumkan ketersediaan **FreeBSD 15.1-RELEASE**. Sebagai rilis kedua dari cabang `stable/15`, pembaruan ini membawa sederet penyempurnaan yang siap membuat pengelolaan server dan *workstation* kamu menjadi jauh lebih mulus.

🚀 Sorotan Utama Rilis 15.1

Tim pengembang FreeBSD benar-benar fokus pada modernisasi dan stabilitas sistem. Berikut adalah beberapa hal baru yang wajib kamu tahu:

  • **Pembaruan Driver Nirkabel:** Driver nirkabel `iwlwifi(4)` beserta driver jaringan lain yang berbasis LinuxKPI kini sudah beralih menggunakan dasar Linux v7.0. Ini berarti dukungan perangkat keras jaringan yang jauh lebih baik dan ngebut.
  • **Kemudahan di Cloud:** Kalau kamu sering men- *deploy* instance di cloud, kini *image* cloud FreeBSD yang berbasis terpaket (*packaged base system*) sudah dibekali dengan `pkg(8)`. Kerennya lagi, ini langsung mendukung pembaruan otomatis paket sistem basis sejak booting pertama!
  • **Kustomisasi Penjadwal (Scheduler):** Hadirnya *tunable* baru `kern.sched.name` memberikan kebebasan bagi para *sysadmin* untuk memilih penjadwal kernel langsung pada saat proses booting.
  • **Dukungan Bahasa C Modern:** Sudah ada kemajuan yang sangat signifikan menuju dukungan penuh standar bahasa pemrograman **C23**, sebuah langkah manis untuk komunitas *developer*.
  • **Dukungan Unicode Terkini:** Sistem kini diperbarui untuk mengenali **Unicode 17.0.0** dan **CLDR 48**, menambahkan sekitar 4.803 karakter baru. Bye-bye teks kotak-kotak tidak terbaca!

Tentu saja, detail teknis, daftar kompatibilitas perangkat keras yang lebih rinci, dan daftar *errata* bisa kamu selami langsung di catatan rilis resmi mereka melalui tautan berikut:

  • Catatan Rilis: https://www.FreeBSD.org/releases/15.1R/relnotes/
  • Dukungan Perangkat Keras: https://www.FreeBSD.org/releases/15.1R/hardware/
  • Erata: https://www.FreeBSD.org/releases/15.1R/errata/
  • Aktivitas Rilis (Releng): https://www.FreeBSD.org/releng/

💻 Ketersediaan & Dukungan Arsitektur

Versi terbaru ini tidak pilih kasih. FreeBSD 15.1-RELEASE sudah siap diunduh untuk berbagai macam arsitektur kelas berat maupun ringan, termasuk: **amd64, aarch64, armv7, powerpc64, powerpc64le, dan riscv64**.

Untuk memastikan integritas berkas, *Hash SHA512* dan *SHA256* untuk image ISO, stik memori, dan kartu SD telah disediakan. Kamu juga bisa memverifikasi *checksum* bertanda tangan PGP yang bisa diakses di: https://www.FreeBSD.org/releases/15.1R/signatures/ (sedangkan untuk pesan pengumuman PGP tersedia di sini).

🛠️ Panduan Eksekusi Media Instalasi

Tergantung dari bagaimana kamu akan memasang sistem ini, FreeBSD telah menyediakan bermacam-macam *image* instalasi. Di bawah ini adalah panduan singkat serta contoh cara nge- *burn* image-nya menggunakan terminal (asumsi *flash drive* USB kamu terbaca sebagai `/dev/da0`, **pastikan targetnya benar agar data lain tidak terhapus!**):

1. Image `dvd1`

Ini adalah paket super lengkap! Berisi OS dasar FreeBSD, dokumentasi, *debugging*, dan sekumpulan paket *pra-bangun* untuk kebutuhan *workstation* grafis. Mendukung juga *livefs rescue mode*.

# Menulis ke USB flash drive untuk instalasi amd64
# dd if=FreeBSD-15.1-RELEASE-amd64-dvd1.iso of=/dev/da0 bs=1m conv=sync

2. Image `disc1`

Image klasik untuk sistem operasi dasar. Tidak menyertakan paket pihak ketiga pra-bangun, tetapi tetap mendukung mode *livefs*.

# Menulis ke USB flash drive untuk instalasi amd64
# dd if=FreeBSD-15.1-RELEASE-amd64-disc1.iso of=/dev/da0 bs=1m conv=sync

3. Image `bootonly`

Sesuai namanya, ini murni hanya untuk melakukan *boot*. Distribusi instalasinya akan ditarik dari jaringan (HTTP/FTP). Sangat cocok untuk *deployment* masal via server lokal.

# Persiapan bootonly di flash drive
# dd if=FreeBSD-15.1-RELEASE-amd64-bootonly.iso of=/dev/da0 bs=1m conv=sync

4. Image `memstick` & `mini-memstick`

Format `.img` yang dirancang khusus dari awal untuk *flash drive* USB. `memstick` berisi set dasar instalasi, sedangkan `mini-memstick` menarik data dari jaringan layaknya versi *bootonly*.

# Menggunakan image memstick standar
# dd if=FreeBSD-15.1-RELEASE-amd64-memstick.img of=/dev/da0 bs=1m conv=sync
# Menggunakan image mini-memstick
# dd if=FreeBSD-15.1-RELEASE-amd64-mini-memstick.img of=/dev/da0 bs=1m conv=sync

5. Image Kartu SD untuk FreeBSD/arm

Punya Raspberry Pi atau board berbasis ARM lainnya? FreeBSD menyediakan image kartu SD yang bisa muat bahkan di kartu sekecil 5 GB. Ingat, *username* bawaan adalah `freebsd` (password: `freebsd`) dan *root* punya password default `root`. **Segera ganti password ini!**

# Ganti 'KERNEL' sesuai arsitektur, contoh untuk RPI:
# dd if=FreeBSD-15.1-RELEASE-arm64-aarch64-RPI.img of=/dev/da0 bs=1m conv=sync

☁️ Ketersediaan di Era Cloud (AWS, GCP, Azure)

Buat kamu yang bermain di ekosistem Cloud, FreeBSD 15.1 sudah nangkring dengan cantik di platform raksasa dunia dalam format QCOW2, VHD, VMDK, dan *raw image*.

Amazon EC2

Kunci Parameter Store AWS (SSM) telah disediakan untuk memudahkan pencarian AMI **FreeBSD/amd64** dan **FreeBSD/aarch64** (mendukung format UFS maupun ZFS). Kamu bisa mencari kunci seperti `/aws/service/freebsd/amd64/base/zfs/15.1/RELEASE` langsung di tiap region AWS. Selain itu, *image* ini juga tersedia langsung di AWS Marketplace.

Google Compute Engine (GCP)

Buat instance baru semudah mengetik dua baris perintah gcloud berikut (jangan lupa ganti `INSTANCE` dengan nama servermu):

% gcloud compute instances create INSTANCE --image freebsd-15-1-release-amd64-ufs --image-project=freebsd-org-cloud-dev
% gcloud compute ssh INSTANCE

Microsoft Azure

Pengguna layanan Microsoft tidak perlu gigit jari, karena FreeBSD 15.1-RELEASE dapat segera diputar dari Azure Marketplace.

⏳ Linimasa Dukungan (Support Lifecycle)

Menjalankan server berarti bicara soal dukungan jangka panjang. Rilis **FreeBSD 15.1-RELEASE** ini akan mendapat dukungan keamanan penuh hingga **31 Maret 2027**. Sebagai catatan, rilis 15.0-RELEASE akan *End of Life* pada 30 September 2026. Secara keseluruhan, keluarga besar FreeBSD 15 ini akan terus didukung hingga **31 Desember 2029**. Santai kan?
