Perbaikan Otomatis GRUB Bootloader

Jika GRUB Bootloader rusak, sistem Debian atau turunannya tidak akan bisa booting. Skrip ini akan melakukan proses **mount → chroot → reinstall GRUB** secara otomatis. Pastikan Anda tahu disk target yang benar (misalnya `/dev/sda`).

Skrip Otomatis (`fix-grub.sh`)

#!/bin/bash
# fix-grub.sh - Skrip perbaikan GRUB otomatis
# Peringatan: Pastikan disk target sudah benar!
DISK="$1"
if [ -z "$DISK" ]; then
  echo "Penggunaan: sudo $0 /dev/sdX"
  exit 1
fi
mount "${DISK}1" /mnt
for i in /dev /proc /sys; do mount --bind "$i" "/mnt$i"; done
chroot /mnt grub-install "$DISK"
chroot /mnt update-grub
for i in /sys /proc /dev; do umount "/mnt$i"; done
umount /mnt
echo "GRUB berhasil diperbaiki. Silakan reboot."

Cara Menggunakan

  1. Simpan skrip di atas sebagai `fix-grub.sh`.
  2. Boot ke Live USB Debian/Ubuntu dan buka terminal.
  3. Beri izin eksekusi: `chmod +x fix-grub.sh`
  4. Jalankan dengan nama disk Anda: `sudo ./fix-grub.sh /dev/sda`
