Google Sans Flex: Apa Kelebihannya? 🤔

Dirilis di bawah lisensi SIL Open Font License, Google Sans Flex adalah font generasi terbaru Google yang didesain oleh David Berlow. Istilah kerennya, ini adalah Variable Font.

Apa itu font variabel? Bayangin kamu cuma butuh satu file font, tapi di dalamnya sudah mencakup lima "poros" (axes) yang bisa diatur sesuka hati: ketebalan (weight), lebar (width), ukuran optik, kemiringan, sampai seberapa bulat ujung hurufnya. Jadi, satu file mungil ini bisa menggantikan puluhan file font tradisional yang bikin folder .fonts kamu penuh sesak!

Google bikin font ini khusus buat layar modern dengan berbagai ukuran. Jadi, kalau kamu pake monitor resolusi tinggi (4K) dengan fractional scaling, font ini bakal tetep kelihatan tajam, tidak pecah atau "blur" kayak hubungan kamu sama dia. Haha!

📥 Cara Pasang Google Sans Flex di Linux

Langkahnya gampang banget:

  1. Download file font-nya dari web resmi Google Fonts.
  2. Ekstrak filenya.
  3. Klik dua kali file .ttf-nya lalu klik "Install", atau pindahin ke folder ~/.local/share/fonts/.
  4. Gunakan GNOME Tweaks atau tool bawaan distro kamu buat ganti font sistem.
