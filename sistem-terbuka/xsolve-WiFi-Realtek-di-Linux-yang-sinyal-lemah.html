Fix WiFi Realtek di Linux (Sinyal Lemah)

Jika kamu menggunakan adapter WiFi Realtek di Linux dan mengalami sinyal lemah, beberapa perintah **iw** dan **modprobe** dapat membantu memperbaikinya. Pastikan kamu menjalankan perintah ini dengan hak akses `sudo`.

# Cek interface WiFi
iwconfig
# Hentikan interface yang bermasalah
sudo ifconfig wlan0 down
# Muat ulang modul dengan opsi optimasi
sudo modprobe -r rtl8723be
sudo modprobe rtl8723be ant_sel=2
# Aktifkan kembali interface
sudo ifconfig wlan0 up
# Scan jaringan WiFi
sudo iwlist wlan0 scan

Penjelasan Perintah

**iwconfig**: Melihat interface wireless dan statusnya.

**ifconfig wlan0 down/up**: Menonaktifkan/aktifkan kembali interface WiFi.

**modprobe -r rtl8723be**: Melepaskan modul kernel WiFi Realtek.

**modprobe rtl8723be ant_sel=2**: Memuat ulang modul dengan opsi antenna selection, sering meningkatkan sinyal.

**iwlist wlan0 scan**: Memindai jaringan WiFi yang tersedia.
