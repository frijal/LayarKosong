Membagikan Aplikasi Debian dengan dpkg-repack

Banyak pengguna Debian rajin membersihkan cache APT, tapi bagaimana jika ingin berbagi aplikasi tanpa koneksi internet? Solusinya sederhana: gunakan `dpkg-repack`.

Bayangkan Anda ingin memperlihatkan aplikasi baru di laptop, lalu teman Anda juga ingin memasangnya. Tidak ada internet, tidak bawa repositori, hanya ada flashdisk kosong. Di sinilah `dpkg-repack` menjadi pahlawan.

Langkah-langkah Menggunakan dpkg-repack

Prosesnya sangat mudah dan hanya memerlukan beberapa perintah di terminal.

1. Instalasi `dpkg-repack`

Pastikan paket `dpkg-repack` sudah terpasang. Jika belum, Anda perlu koneksi internet untuk menginstalnya sekali ini saja.

sudo apt-get install dpkg-repack

2. Buat Ulang Paket .deb

Gunakan perintah `dpkg-repack` diikuti dengan nama paket yang ingin Anda bagikan. Contoh, untuk memaketkan ulang pemutar musik **Amarok**:

dpkg-repack amarok

Setelah perintah dijalankan, file `amarok_VERSION_ARCH.deb` akan muncul di direktori Anda saat ini. Cukup salin file .deb tersebut ke flashdisk dan bagikan ke teman Anda!

Tips: Menggunakan Fakeroot

Jika Anda ingin menjalankan perintah tanpa `sudo`, Anda bisa menggunakan bantuan **fakeroot**. Ini berguna untuk menghindari masalah perizinan.

fakeroot -u dpkg-repack nama-paket
