âš¡ 2. Upgrade Paralel dengan apt-fast

Biasanya apt hanya mengunduh 1 file per waktu. Nah, dengan apt-fast kamu bisa unduh paket paralel menggunakan beberapa koneksi, jadi upgrade makin kilat!

  1. Instalasi apt-fast:
    • sudo add-apt-repository ppa:apt-fast/stable
    • sudo apt update
    • sudo apt install apt-fast
  2. Saat instalasi, kamu akan diminta memilih metode (apt/apt-get/aptitude) dan jumlah koneksi (default 5 aja sudah ngebut).
  3. Mulai gunakan apt-fast sebagai ganti sudo apt upgrade atau sudo apt install.

ðŸ§  3. Preload Aplikasi Favorit ke RAM

Punya aplikasi yang sering banget dipake? Biar makin satset, install preload supaya aplikasi favorit kamu otomatis dimuat ke RAM saat booting atau saat kamu butuh.

  1. Instalasi: sudo apt install preload
  2. Tidak perlu konfigurasi manual, preload akan berjalan otomatis di latar belakang.
  3. Cek status atau log untuk memastikan berjalan: cat /var/log/preload.log

ðŸ’¾ 4. Optimasi SSD dengan fstrim

Jika storage kamu SSD, wajib aktifkan layanan fstrim (TRIM command) supaya ruang kosongnya makin efisien (wear leveling) dan SSD lebih awet dan cepat.

Ubuntu modern harusnya sudah mengaktifkannya secara default melalui fstrim.timer, tapi tidak ada salahnya mengecek:

  1. Cek status: sudo systemctl status fstrim.timer (Harusnya aktif/enabled)
  2. Aktifkan (jika belum):
    • sudo systemctl enable fstrim.timer
    • sudo systemctl start fstrim.timer
  3. Trim manual (opsional): sudo fstrim -av (melakukan trim pada semua mount points yang mendukung)
