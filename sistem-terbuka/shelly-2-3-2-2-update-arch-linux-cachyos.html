Shelly 2.3.2.2 Rilis: Update Stabil dengan Dukungan CachyOS dan AppImage Drag-and-Drop

Berdasarkan rilis resmi Seafoam Labs 7 Juni 2026 Arch Linux, CachyOS, Package Manager

Pengembang **Zoey Bauer** merilis **Shelly 2.3.2.2** hari ini sebagai update stabil terbaru untuk graphical package manager open-source yang ditujukan bagi distribusi berbasis Arch Linux. Update ini membawa peningkatan signifikan, terutama bagi pengguna **CachyOS**, sekaligus memperkenalkan cara baru yang lebih praktis untuk mengelola aplikasi AppImage.

Shelly sendiri merupakan reimajinasi modern dari package manager default Arch Linux. Selain mendukung AUR dan Flathub, kini ia juga semakin tangguh dengan dukungan AppImage secara native. **CachyOS baru-baru ini mengadopsi Shelly** sebagai GUI package manager default mereka, menandai langkah besar bagi proyek ini.

Peningkatan Dukungan untuk CachyOS

Shelly 2.3.2.2 secara khusus meningkatkan kompatibilitas dengan distribusi **CachyOS**. Jalur updater CachyOS kini memiliki fitur parity yang lebih lengkap. Pengguna juga dapat menurunkan versi package langsung dari repositori CachyOS menggunakan perintah baru yang disediakan.

Yang paling memudahkan, Shelly kini **secara otomatis mendeteksi repositori pilihan** pengguna. Tidak perlu lagi konfigurasi manual yang rumit — cukup jalankan perintah dan Shelly akan mengenali konteks distribusi yang sedang digunakan.

Instalasi AppImage yang Lebih Simpel

Salah satu fitur paling disambut baik adalah kemampuan menginstal aplikasi AppImage melalui **drag-and-drop**. Cukup seret file AppImage ke jendela Shelly, maka proses instalasi akan berjalan otomatis tanpa perlu perintah terminal yang panjang.

Selain itu, rilis ini menambahkan:

  • Dukungan command-line untuk mengisi **gpg keyring** dengan lebih mudah
  • Sorting berdasarkan kolom ukuran package di tampilan daftar
  • Dukungan flag **--now** untuk berbagai operasi sistem
  • Kemampuan mengurai berita Arch Linux dari format HTML menjadi Markdown yang lebih nyaman dibaca

**Catatan praktis:** Fitur-fitur baru ini dirancang agar pengguna Arch Linux dan turunannya bisa bekerja lebih cepat tanpa perlu sering berpindah antara GUI dan terminal. Hasil dapat berbeda tergantung perangkat, versi paket, dan konfigurasi sistem.

Penyempurnaan Antarmuka Pengguna

Tim Shelly juga melakukan banyak sentuhan pada sisi visual dan pengalaman pengguna. Halaman **Recommendations** kini dilengkapi loading state yang lebih jelas serta kotak deskripsi yang diperbaiki tampilannya.

Dialog pertanyaan menampilkan daftar package terinstal dengan visual yang lebih rapi. Baris di **Shelly Search** menjadi lebih compact sehingga lebih banyak informasi yang bisa ditampilkan dalam satu layar. Margin dan alignment pada dialog detail package juga diperbaiki agar terasa lebih profesional dan nyaman digunakan.

Perbaikan Bug dan Update Terjemahan

Beberapa bug yang mengganggu telah diperbaiki, antara lain masalah pada proses downgrade package serta kegagalan fungsi **“upgrade all”** ketika terjadi partial failure pada daftar upgrade.

Di sisi multibahasa, ditambahkan terjemahan baru untuk **bahasa Jerman, Hungaria, dan Jepang**. Sementara terjemahan untuk **Polandia, Brazilian Portuguese, dan Catalan** juga diperbarui agar lebih akurat dan natural.

Di Balik Layar: Stabilitas dan Keandalan

Seperti biasa pada rilis stabil, tim melakukan pembersihan kode serta berbagai peningkatan di balik layar. Tujuannya sederhana: membuat Shelly semakin andal dan minim gangguan saat digunakan sehari-hari.

Semua perubahan ini bisa dilihat secara lengkap di halaman rilis resmi proyek.

Untuk mencoba Shelly 2.3.2.2, unduh binary siap pakai atau source code langsung dari halaman rilis GitHub Shelly. Update ini tersedia untuk semua distribusi berbasis Arch Linux.
