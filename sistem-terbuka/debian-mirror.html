Skrip Debian-Mirror.sh: Pilih Mirror Tercepat

Pernah tidak sih kamu merasa kesal karena proses sudo apt update atau instalasi paket di Debian maupun Ubuntu terasa sangat lambat? Sering kali, biang keladinya bukan internetmu yang bermasalah, melainkan server mirror repository yang sedang kamu gunakan terlalu jauh atau sedang penuh sesak oleh trafik.

Kondisi ini bikin waktu produktifmu terbuang sia-sia hanya untuk mengunduh daftar paket. Beruntungnya, ada cara cerdas untuk mengatasi hal ini dengan bantuan utilitas netselect-apt. Kita akan mengotomatisasi pemilihannya agar sistem selalu mengambil data dari server yang paling "ngebut" buat lokasi tempat tinggalmu.

Ingat ya, sebelum menjalankan langkah-langkah di bawah ini, pastikan kamu memiliki akses sudo, karena sistem akan mengubah file konfigurasi inti pada /etc/apt/sources.list.

Implementasi Skrip Otomatis

Agar proses ini tidak merepotkan, kita bisa membungkus logika pemindaian dan penggantian mirror ke dalam sebuah skrip bash. Kamu hanya perlu menyalin kode di bawah ini ke dalam file, beri izin akses, dan jalankan.

#!/bin/bash
# debian-mirror.sh - Otomatis pilih mirror tercepat untuk Debian/Ubuntu
set -e
echo "[INFO] Memperbarui daftar paket & menginstal netselect-apt..."
sudo apt update
sudo apt install -y netselect-apt
echo "[INFO] Menjalankan netselect-apt untuk rilis 'stable'..."
sudo netselect-apt stable
echo "[INFO] Mem-backup sources.list yang lama..."
sudo cp /etc/apt/sources.list /etc/apt/sources.list.bak
echo "[INFO] Mengganti sources.list dengan hasil mirror tercepat..."
sudo mv sources.list /etc/apt/sources.list
echo "[INFO] Memperbarui daftar paket dari mirror baru..."
sudo apt update
echo "[SELESAI] Mirror tercepat berhasil dipasang!"

Skrip ini bekerja dengan cara menguji latensi (waktu respon) dari berbagai cermin repositori di seluruh dunia dan secara pintar menentukan mana yang paling dekat serta responsif dengan koneksi internet yang kamu pakai saat ini.

Proses Eksekusi

Setelah file debian-mirror.sh tersimpan di direktori kerja kamu, berikan izin akses eksekusi agar skrip dapat berjalan:

chmod +x debian-mirror.sh
sudo ./debian-mirror.sh

Sistem akan secara otomatis menjalankan proses pengujian mirror. Tunggu beberapa saat sampai terminal memberikan notifikasi bahwa proses telah selesai. Setelah itu, kamu bisa mencoba melakukan instalasi aplikasi dan rasakan perbedaannya. Selamat menikmati kecepatan unduh yang lebih stabil dan responsif!
