Skrip quick-mirror.sh: Optimasi Server Debian & Arch

Pernahkah kamu duduk bengong memperhatikan layar terminal yang merayap di angka beberapa kilobyte per detik saat sedang melakukan update sistem? Apalagi kalau lagi nyantai di Balikpapan sambil seruput kopi, rasanya waktu terbuang sia-sia hanya untuk menunggu baris unduhan selesai. Drama koneksi lambat ini umumnya bukan salah penyedia internet, melainkan rute repositori bawaan Linux yang memutar terlalu jauh secara geografis.

Bagi kamu para pengguna Debian, Ubuntu, maupun Arch Linux yang dinamis, konfigurasi mirror default sering kali tidak optimal. Server utama mungkin berada di belahan benua lain, yang otomatis meningkatkan latensi dan menurunkan kecepatan transfer. Itulah sebabnya, memiliki alat cerdas untuk mencari dan mematok mirror terdekat yang sehat adalah sebuah keharusan mutlak.

Solusi All-in-One: Skrip quick-mirror.sh

Daripada repot menghapal dan mengetik perintah secara manual setiap kali kecepatan unduhan mulai kembang kempis, kamu bisa meringkas semuanya ke dalam satu bash script serbaguna bernama quick-mirror.sh. Skrip ini didesain sebagai wrapper pintar yang akan mendeteksi argumen input kamu.

Jika kamu memberikan argumen debian, ia akan otomatis memanggil utilitas netselect-apt untuk menguji ping server stable dan mengganti konfigurasi lawas kamu secara bersih. Namun, jika kamu memasukkan arch, ia langsung berbelok menggunakan reflector untuk menjaring 10 mirror lokal paling baru (khususnya wilayah Indonesia) dan mengurutkannya berdasarkan kecepatan aktual. Sangat praktis, bukan?

⚠️ Peringatan Akses: Skrip ini wajib dijalankan dengan memberikan kekuasaan superuser (sudo). Skrip akan melakukan pencadangan otomatis (backup) dan menimpa file krusial sistem di dalam direktori /etc/apt/ maupun /etc/pacman.d/.

Kode Skrip Inti

Cara pemakaiannya gampang banget. Silakan salin blok kode di bawah ini, simpan dengan nama quick-mirror.sh, dan jangan lupa berikan hak eksekusi dengan perintah chmod +x quick-mirror.sh di terminal kesayanganmu.

#!/bin/bash
# quick-mirror.sh - Pilih mirror tercepat untuk Debian/Ubuntu atau Arch
# Penggunaan: sudo ./quick-mirror.sh [debian|arch]
MODE="$1"
if [ "$MODE" = "debian" ]; then
 echo "Mode Debian/Ubuntu dipilih..."
 sudo apt update
 sudo apt install -y netselect-apt
 sudo netselect-apt stable
 sudo cp /etc/apt/sources.list /etc/apt/sources.list.bak
 sudo mv sources.list /etc/apt/sources.list
 sudo apt update
elif [ "$MODE" = "arch" ]; then
 echo "Mode Arch Linux dipilih..."
 sudo pacman -S --noconfirm reflector
 sudo reflector --country "Indonesia" --latest 10 --sort rate --save /etc/pacman.d/mirrorlist
 sudo pacman -Syyu
else
 echo "Penggunaan tidak valid."
 echo "Contoh: $0 debian"
 echo "atau: $0 arch"
fi

Setelah menjalankan skrip tersebut (contoh: sudo ./quick-mirror.sh debian), kamu bisa langsung menikmati aliran bandwidth maksimal. Kini, memelihara dan merawat kesehatan repositori sistem kamu bukan lagi beban, melainkan hiburan ringan yang bisa diselesaikan dalam hitungan detik. Informasi ini bersifat edukatif dan tidak menggantikan saran tenaga kesehatan.
