Panduan Restore Data Pakai Deja Dup dan Timeshift di Debian

Buat kamu yang pasti paham rasanya deg-degan habis install ulang server atau laptop Debian kesayangan. Install ulang itu memang bikin sistem jadi segar, tapi urusan mengembalikan (restore) data kadang bikin pusing. Apalagi kalau disuruh melototin layar hitam terminal sambil mengetik kode yang panjang-panjang. Hadeh, bikin lelah mata!

Tenang saja, kawan. Kita tidak perlu umumnya menjadi pelaku ancaman siber yang kaku. Pastikan seluruh langkah dilakukan hanya pada perangkat sendiri atau lingkungan yang memiliki izin resmi. Berikut adalah panduan menggunakan dua aplikasi Graphical User Interface (GUI) yang sangat memanjakan mata untuk mengembalikan data kamu: Deja Dup dan Timeshift. Keduanya punya fungsi yang sedikit berbeda, mari kita bedah satu per satu!

1. Restore Menggunakan Deja Dup (GUI) — Paling Direkomendasikan

Jika tujuan utamamu murni ingin menyelamatkan dokumen pribadi, tugas kuliah, koleksi foto meme, dan file unduhan, maka Deja Dup adalah sang juru selamat. Aplikasi ini sering kali sudah terpasang bawaan di banyak distro berbasis Debian atau Ubuntu. Antarmukanya sangat sederhana sehingga ramah bagi pemula.

Langkah-langkah Eksekusi Deja Dup

Langkah 1: Install Deja Dup
Buka Terminal kesayanganmu sebentar saja dan jalankan:

sudo apt update
sudo apt install deja-dup -y

Langkah 2: Buka Aplikasinya
Cari ikon aplikasi bernama "Backups" di menu deretan aplikasi kamu, atau cukup panggil lewat terminal dengan mengetik deja-dup.

Langkah 3: Tentukan Lokasi Harta Karunmu
Di jendela aplikasi, klik tabRestore. Setelah itu, tekan tombol tebal bertuliskan "Restore from a previous backup". Arahkan ke flashdisk atau HDD eksternal tempat kamu menyimpan data lamamu, lalu klik Forward.

Langkah 4: Pilih Data yang Ingin Dikembalikan
Aplikasi canggih ini akan membaca arsipmu dan menampilkan struktur folder layaknya file manager biasa. Centang folder-folder krusial seperti Documents, Pictures, Downloads, atau .config (jika kamu juga menyimpan pengaturan dotfiles). Setelah yakin, klik Forward lagi.

Langkah 5: Tentukan Lokasi Pendaratan
Sistem akan bertanya di mana data ini mau diletakkan. Sangat disarankan untuk memilih opsi "Restore files to original locations" agar sistem secara otomatis meletakkannya kembali ke /home milikmu. Klik Restore.

Langkah 6: Tunggu Sambil Ngopi
Biarkan sistem memindahkan data. Proses ini bisa berjalan beberapa menit hingga hitungan jam tergantung ukuran datamu. Jika sudah selesai, reboot komputermu biar afdal.

2. Restore Menggunakan Timeshift (Mesin Waktu Sistem)

Timeshift memiliki arena bermain yang berbeda. Daripada mengurusi file Excel atau PDF-mu, ia lebih fokus untuk mengambil gambar utuh (snapshot) dari sistem operasimu. Mirip seperti fitur System Restore di Windows.

Peringatan Ringan: Karena kamu berada pada skenario baru selesai install ulang, cara ini hanya bisa dilakukan jika di OS yang lama kamu sempat membuat arsip snapshot Timeshift dan menyimpannya di partisi atau diska eksternal yang terpisah.

Langkah-langkah Eksekusi Timeshift

Langkah 1: Install Timeshift

sudo apt install timeshift -y

Langkah 2: Buka dan Autentikasi
Cari ikon aplikasinya atau jalankan sudo timeshift. Ia butuh akses penuh (root) untuk memanipulasi tulang punggung sistem.

Langkah 3: Pilih Titik Kembalimu
Di halaman utama, klik opsi Restore. Arahkan pembacaan data ke HDD eksternalmu. Pilih snapshot dengan tanggal terbaru sebelum OS lamamu rusak atau diinstal ulang.

Langkah 4: Pilih Mode Target
Timeshift akan menyuguhkan dua menu makanan:

  • Restore system files only (Cuma file sistem root).
  • Restore system + home files (Sistem di-rollback + data pribadi juga ikut diubah).

Pilih sesuai seleramu, lalu klik Next.

Langkah 5: Konfirmasi Maut
Tinjau ulang apa saja partisi yang akan ditimpa. Jika sudah yakin 100%, klik Restore. Ingat, proses ini akan menimpa seluruh sistem barumu kembali ke state saat snapshot diambil.

Langkah 6: Wajib Reboot
Setelah indikator progres penuh, restart komputermu agar kernel Linux memuat sistem dari posisi masa lalunya.

Langkah Pamungkas (Bonus Tip)

Karena baik Deja Dup maupun Timeshift tidak punya kemampuan sakti untuk mengembalikan aplikasi-aplikasi yang terinstal via apt secara mandiri, kamu harus sedikit menggunakan jurus terminal. Selesai memulihkan data, segera kembalikan daftar aplikasimu dan sempurnakan perizinan berkas:

# 1. Suntik ulang daftar package dari backup-mu
sudo dpkg --set-selections < /media/namauser/Backup/installed-packages.txt
sudo apt-get dselect-upgrade -y
# 2. Rapikan kepemilikan file agar tidak gembok-gembokan
sudo chown -R namauser:namauser /home/namauser
# 3. Update dan bersihkan sistem
sudo apt update && sudo apt full-upgrade -y

Nah, selesai sudah! Tidak perlu pusing lagi kan? Semoga Debian barumu awet dan performanya sekencang roket! 🚀
