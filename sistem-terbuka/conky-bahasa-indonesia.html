Kalau kamu merasa desktop Linux kamu terlalu sepi, **Conky** adalah obatnya. Bayangkan punya panel transparan ala film fiksi ilmiah yang nunjukin kecepatan internet atau beban CPU secara real-time. Di panduan ini, aku sudah racikkan konfigurasi khusus yang pakai **Bahasa Indonesia**, biar lebih akrab di mata kita orang Balikpapan dan sekitarnya!

Tahap 1: Instalasi

Pertama, kita butuh mesin utamanya. Jangan khawatir, Conky sangat ringan dan tersedia di hampir semua distro.

# Pilih sesuai distro andalanmu:
sudo apt install conky-all       # Debian/Ubuntu/Mint
sudo pacman -S conky             # Arch/Manjaro
sudo dnf install conky           # Fedora

Tahap 2: Script Anti-Error

Seringkali Conky muncul duluan sebelum wallpaper siap, akhirnya tampilannya jadi berantakan (glitch). Solusinya, kita buat skrip jeda 10 detik.

# Buat file: ~/.mulaiconky.sh
#!/bin/bash
sleep 10 && conky -c "$HOME/.conkyrc" &
Penting: Jangan lupa jalankan `chmod +x ~/.mulaiconky.sh` biar bisa dieksekusi!

Tahap 3: Konfigurasi .conkyrc

Ini adalah bagian intinya. Copy kode di bawah ini ke dalam file `~/.conkyrc` di home directory kamu.
