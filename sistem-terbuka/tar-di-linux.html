Tar di Linux: Kompres & Extract Santai dari Terminal

Mau mengompres dan mengekstrak file arsip dengan mudah di Linux? Gunakan perintah `tar`â€”langsung dari terminal, fleksibel, dan cepat. Cocok untuk backup, deployment, hingga distribusi proyek.

Perintah Dasar

Membuat Arsip `.tar.gz`

tar -czf nama_arsip.tar.gz folder_atau_file/

Mengekstrak Arsip `.tar.gz`

tar -xzf nama_arsip.tar.gz

Variasi Berguna Lainnya

Melihat Isi Arsip Tanpa Ekstrak

tar -tf nama_arsip.tar.gz

Membuat Backup dengan Nama Bertanggal

tar -czf backup-$(date +%Y-%m-%d).tar.gz /path/to/data/

Mengompres Sambil Mengabaikan File Tertentu

tar -czf arsip.tar.gz --exclude="*.tmp" --exclude="cache/" folder/

Penjelasan Opsi (Parameter)

  • `-c`: **Create**, untuk membuat arsip baru.
  • `-x`: **Extract**, untuk mengekstrak isi arsip.
  • `-z`: Menggunakan kompresi **gzip** (menghasilkan `.tar.gz`).
  • `-f`: **File**, untuk menentukan nama file arsip.
  • `-v`: **Verbose**, menampilkan proses file yang sedang dikerjakan.
  • `-t`: **List**, untuk melihat daftar isi arsip.

Alternatif Lain

Selain `tar`, Anda juga bisa menggunakan `zip` atau `7z` yang mungkin lebih familiar bagi pengguna Windows.

  • **zip:** `zip -r arsip.zip folder/`
  • **7z:** `7z a arsip.7z folder/` (biasanya kompresi lebih baik)
