1 — Training tentang Baris Perintah (Command Line) Linux

Pengantar singkat dan manfaat menguasai CLI.

Command Line (baris perintah) adalah antarmuka berbasis teks untuk berinteraksi langsung dengan sistem operasi. Menguasainya memberi Anda kontrol cepat, kemampuan scripting, dan opsi otomatisasi yang tidak mudah dicapai via GUI.

**Manfaat utama:**

  • Efisiensi: tugas berulang dapat diotomasi.
  • Power user: akses fitur lanjutan, konfigurasi, dan debugging.
  • Resource-light: dapat dijalankan di sistem minimal.

Ilustrasi: Terminal Linux siap menerima perintah.

2 — Definisi singkat tentang Baris Perintah (Command Line)

Apa itu command line dan kapan menggunakannya? Command line (CLI) adalah antarmuka teks yang menerima perintah, mengeksekusi, lalu menampilkan hasil. Contoh: membuka terminal dan mengetik `ls`, `cd`, atau menjalankan skrip.

3 — `man` (Manual)

Fungsi: membaca manual / dokumentasi perintah.

`man` menampilkan manual (manpage) untuk perintah atau program. Manpages adalah sumber rujukan primer yang menjelaskan sintaks, opsi, contoh, dan bagian lain.

**Bagian umum di manpage:**

  1. **NAME** – nama perintah dan ringkasan
  2. **SYNOPSIS** – cara pemanggilan
  3. **DESCRIPTION** – penjelasan fungsi
  4. **OPTIONS** – daftar opsi
  5. **EXAMPLES** – contoh pemakaian
  6. **SEE ALSO** – referensi terkait
man ls
man -k copy # cari topik (apropos)
**💡 Tips:** Tekan q untuk keluar. Gunakan `man -k subject` untuk mencari perintah. Jika manpage tidak ada, coba `--help`.

4 — `cd` (Change Directory)

Fungsi: berpindah direktori kerja.

Sintaks: `cd [path]`. Path bisa absolut atau relatif. Shortcut: `cd` saja ke home, `cd ~`, `cd -` ke direktori sebelumnya, `cd ..` naik satu level.

cd /home/user/Documents
cd .. # naik satu tingkat
cd - # kembali ke direktori sebelumnya
**🚀 Tips:** Gunakan Tab untuk auto-complete nama direktori. Di skrip, pakai `pushd`/`popd`.

5 — `ls` (Listing)

Fungsi: menampilkan isi direktori.

Opsi umum: `-l` long listing, `-a` tampilkan file tersembunyi, `-h` ukuran mudah dibaca, `-R` rekursif, `-t` urut waktu.

ls -lah
ls -l --color=auto
ls -Rt

**Membaca output `ls -l`:** `-rw-r--r-- 1 user group 2048 ...` : tipe file, permission, link count, ukuran.

6 — `mkdir` (Make Directory)

Fungsi: membuat direktori baru.

`mkdir -p` membuat parent direktori sekaligus, `-m` set permission.

mkdir proyek_baru
mkdir -p ~/projects/2025/aplikasi
**💡 Catatan:** Permission dipengaruhi `umask`. Gunakan `-p` untuk subfolder nested tanpa error.

7 — `ln` (Link)

Fungsi: membuat hard link atau symbolic link.

**Hard link** menunjuk inode sama; **symlink** adalah file pointer.

# hard link
ln asli.txt salinan_hard.txt
# symbolic link
ln -s /path/to/asli.txt symlink.txt

Symlink sering dipakai untuk versi konfigurasi (mis. `/usr/bin/python -> python3`).

8 — `pwd` (Print Working Directory)

Fungsi: menampilkan direktori kerja saat ini.

pwd
pwd -P # physical (follow symlinks)

9 — `cp` (Copy)

Fungsi: menyalin file dan direktori.

Opsi: `-r` rekursif, `-a` archive, `-i` interaktif, `-u` update saja.

cp file1.txt file2.txt
cp -r folder1 backup/folder1
cp -a /etc/skel /backup/skel
**💡 Tips:** Untuk sinkronisasi folder besar, `rsync -av` lebih efisien.

10 — `rm` (Remove)

Fungsi: menghapus file atau direktori.

Hati-hati dengan `rm -rf`. Selalu cek path!

rm file.txt
rm -r folder_contoh
rm -i penting.txt
**⚠️ Keamanan:** Gunakan `trash-cli` untuk pemulihan lebih aman.

11 — `more`

Fungsi: menampilkan file panjang per halaman.

more /var/log/syslog

**💡 Catatan:** `less` lebih fleksibel, tapi `more` masih ada di sistem minimal.

12 — `less`

Fungsi: pager lebih canggih dari `more`.

  • Scroll atas/bawah, pencarian dengan /, mode follow `less +F file`.
less /var/log/syslog
command | less

`less -N` untuk nomor baris. Langsung bisa membuka output perintah: `ps aux | less`.

13 — `touch`

Fungsi: membuat file kosong atau memperbarui timestamp.

touch file_baru.txt # buat file kosong
touch -c file.txt # tidak membuat jika tidak ada
touch -t 202509150830 file.txt # set waktu modifikasi

14 — `mv` (Rename)

Fungsi: mengganti nama file/direktori.

mv lama.txt baru.txt # rename file

Jika di filesystem sama, rename instan. Gunakan `-i` untuk konfirmasi.

15 — `mv` (Move)

Fungsi: memindahkan file/direktori ke lokasi lain.

mv file.txt /home/user/Documents/ # move file

Antar filesystem bisa copy+delete. Gunakan `-v` untuk verbose.

16 — Shortcut Terminal (Ctrl+D, Ctrl+U, Ctrl+L, Ctrl+C, Ctrl+Z)

Kombinasi keyboard penting saat bekerja di terminal.

  • Ctrl+D — EOF, biasanya logout/close shell.
  • Ctrl+U — hapus baris dari kursor ke awal.
  • Ctrl+L — bersihkan layar (clear).
  • Ctrl+C — kirim SIGINT, hentikan proses.
  • Ctrl+Z — suspend proses, lanjutkan dengan `fg`/`bg`.

Ctrl+C: menghentikan proses

Ctrl+Z: proses disuspend

Ctrl+L: clear screen

Ctrl+U: hapus sebelum kursor

**💪 Praktik:** Latih kombinasi ini saat menjalankan proses interaktif (`tail -f`, build) untuk troubleshooting lebih cepat.

17 — Penutup

Ringkasan & saran praktik.

Menguasai perintah dasar (man, cd, ls, mkdir, ln, pwd, cp, rm, more, less, touch, mv) dan shortcut akan membuatmu lebih siap di lingkungan Linux. Latihan rutin dan rajin membaca `man` adalah kuncinya — *RTMF* (Read The Fantastic Manual).

Kalau ada yang ingin ditanyakan atau butuh file PDF/HTML siap unduh, tinggal sampaikan saja ya. Selamat belajar! 🎉
