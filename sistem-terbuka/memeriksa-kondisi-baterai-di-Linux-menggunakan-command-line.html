Memeriksa Kondisi Baterai di Linux

Linux menyediakan beberapa utilitas command line yang sangat berguna untuk memeriksa kondisi baterai laptop Anda. Ini memungkinkan Anda melihat detail seperti kapasitas, status pengisian, dan kesehatan baterai secara keseluruhan.

1. Menggunakan `upower`

`upower` adalah alat standar untuk manajemen daya. Ini memberikan informasi yang sangat detail.

# Menampilkan info detail baterai (ganti BAT0 jika perlu)
upower -i /org/freedesktop/UPower/devices/battery_BAT0

2. Menggunakan `acpi`

`acpi` adalah perintah yang lebih sederhana untuk mendapatkan status baterai dengan cepat.

# Menampilkan status dan persentase baterai
acpi -V

3. Mengakses File Sistem `/sys`

Anda juga bisa melihat informasi baterai secara langsung dari file sistem yang disediakan oleh kernel Linux.

# Menampilkan persentase kapasitas baterai
cat /sys/class/power_supply/BAT0/capacity
# Menampilkan status (misal: Charging, Discharging)
cat /sys/class/power_supply/BAT0/status
