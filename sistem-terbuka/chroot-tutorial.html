Tutorial CHROOT di Debian

chroot adalah perintah di Linux yang berfungsi untuk mengganti direktori root sebuah proses ke direktori lain. Ini memungkinkan kita membuat "ruang isolasi" di mana aplikasi berjalan seolah-olah berada di sistem terpisah.

Skrip Otomatis: Setup & Cleanup

Skrip ini memudahkan proses mount dan unmount saat Anda perlu masuk ke lingkungan chroot, misalnya untuk perbaikan sistem.

#!/bin/bash
# Skrip untuk masuk dan keluar dari chroot dengan mudah
CHROOT_DIR="/mnt" # Ganti jika perlu
enter_chroot() {
    mount --bind /dev "$CHROOT_DIR/dev"
    mount --bind /dev/pts "$CHROOT_DIR/dev/pts"
    mount -t proc proc "$CHROOT_DIR/proc"
    mount -t sysfs sys "$CHROOT_DIR/sys"
    chroot "$CHROOT_DIR"
}
exit_chroot() {
    umount "$CHROOT_DIR/dev/pts"
    umount "$CHROOT_DIR/dev"
    umount "$CHROOT_DIR/proc"
    umount "$CHROOT_DIR/sys"
}
# Contoh penggunaan:
# 1. Mount partisi root Anda: sudo mount /dev/sda1 /mnt
# 2. Masuk chroot: enter_chroot
# 3. Setelah selesai, ketik 'exit'
# 4. Keluar chroot: exit_chroot
