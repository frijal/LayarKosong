Skrip Otomatis Update & Clean APT

Menjaga server atau desktop Linux agar tetap up-to-date adalah sebuah kewajiban absolut, baik demi alasan keamanan maupun performa. Namun, mari kita jujur, mengetik runtutan perintah yang sama secara manual setiap minggu—terlebih saat Anda sedang asyik menikmati kopi di pagi hari, maka pekerjaan itu bisa menjadi rutinitas yang sangat membosankan.

Ketika Anda mengelola mesin berbasis Debian atau Ubuntu, proses pemeliharaan ini tidak sekadar tentang memperbarui perangkat lunak. Ia juga melibatkan pembersihan sisa-sisa paket lama yang dibiarkan menumpuk, memenuhi ruang penyimpanan (disk space), dan membebani sistem Anda. Untuk mengatasi kerepotan repetitif ini, berikut adalah skrip shell sederhana untuk memudahkan proses pembaruan dan pembersihan paket yang tidak terpakai di sistem berbasis Debian/Ubuntu secara komprehensif dalam satu tarikan napas.

⚠️ Catatan: Skrip ini membutuhkan akses sudo. Pastikan Anda menjalankannya dengan hak akses root atau sebagai user dengan izin sudo yang sah agar proses modifikasi repositori tidak terbentur kendala perizinan.

Skrip Lengkap

Alur kerja skrip ini sangat terstruktur. Pertama, ia akan menghubungi server repositori resmi untuk mengecek daftar rilis terbaru (update). Selanjutnya, proses upgrade dieksekusi secara agresif namun aman (berkat flag -y). Setelah pembaruan usai, skrip beralih ke fase "bersih-bersih" di mana ia mencabut dependensi yatim piatu (autoremove) dan membersihkan cache unduhan lama (autoclean & clean).

Cukup salin seluruh baris kode di bawah ini, simpan dengan nama auto-update.sh, kemudian jangan lupa untuk memberikan izin eksekusi lewat perintah chmod +x auto-update.sh sebelum menjalankannya. 

#!/bin/bash
# Skrip untuk otomatis update & clean sistem APT

echo "=== Memperbarui daftar paket ==="
sudo apt update -y

echo "=== Meng-upgrade paket terpasang ==="
sudo apt upgrade -y

echo "=== Menghapus paket yang tidak diperlukan lagi ==="
sudo apt autoremove -y

echo "=== Membersihkan cache APT ==="
sudo apt autoclean -y
sudo apt clean -y

echo "=== Selesai! Sistem sudah diperbarui dan dibersihkan. ==="

Dengan mengotomatisasi rutinitas ini, Anda kini memiliki lebih banyak waktu untuk berfokus pada pekerjaan inti alih-alih menjadi "tukang bersih-bersih" terminal setiap akhir pekan. Jika Anda menginginkan automasi tahap lanjut, Anda bahkan bisa mendaftarkan skrip ini ke dalam penjadwal Cron job untuk berjalan otomatis setiap minggu malam!
