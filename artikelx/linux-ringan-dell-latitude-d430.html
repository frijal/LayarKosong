Fixing the Wi-Fi Problem (Common on Old Dell Laptops)

Your D430 usually has one of these:

  • Broadcom BCM4311
  • Broadcom BCM4312
  • Intel PRO/Wireless 3945

Broadcom cards often need extra drivers.

Step 1 – Identify the Wi-Fi Card

Run:

lspci | grep -i network

Example output:

Broadcom BCM4311 802.11b/g

or

Intel PRO/Wireless 3945ABG

Step 2 – Install Driver

For Intel cards (easy)

Usually works automatically.

Install firmware:

sudo apt install firmware-iwlwifi

For Broadcom cards

Install:

sudo apt install firmware-b43-installer

Then reboot.

If Wi-Fi Still Doesn't Work

Check if the hardware switch is off.

Old Dell laptops often have:

  • Wi-Fi physical switch
  • Fn + F2

Another Trick for Old Laptops

If Wi-Fi still fails, the easiest solution is a:

USB Wi-Fi adapter with Linux support

Chipsets that work perfectly:

  • Atheros
  • Realtek RTL8812AU
  • Mediatek MT7601

These are about $8–$15.
