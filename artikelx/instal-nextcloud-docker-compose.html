Instal Nextcloud Pakai Docker Compose: Cloud Pribadi, Kendali Penuh! ☁️

Apa Itu Nextcloud? (What)

Singkatnya, Nextcloud adalah alternatif terbaik untuk Google Drive, Dropbox, atau OneDrive. Bedanya, kalau layanan-layanan itu menyimpan datamu di server mereka (dan mungkin "mengintip" isinya), Nextcloud menyimpan data di server milikmu sendiri.

Isinya lengkap: bisa simpan foto, kalender, kontak, sampai kolaborasi dokumen ala Google Docs. Dengan bantuan Docker Compose, kita akan menjalankan dua "mesin" sekaligus: aplikasi Nextcloud itu sendiri dan database MariaDB untuk menyimpan semua data pengaturannya.

Kenapa Pakai Docker Compose? (Why)

Bayangkan kamu mau masak nasi goreng. Kamu butuh nasi (aplikasi) dan bumbu (database). Docker Compose itu kayak resep tertulis yang otomatis menyiapkan semuanya. Cukup tulis satu file konfigurasi, ketik satu perintah, dan booom! Semuanya langsung beres terinstal dan saling terhubung secara otomatis. Nggak ada lagi drama database gagal konek!

Di Mana dan Kapan Kita Mulai? (Where & When)

Tentu saja di server Debian yang sudah kita pasang Docker-nya kemarin. Kamu bisa melakukannya sekarang juga lewat terminal atau SSH. Pastikan kamu berada di direktori yang lega, karena semua file cloud kamu bakal mendarat di sini.

Bagaimana Cara Instalnya? (How)

Ikuti langkah-langkah santai berikut ini, jangan sampai ada yang terlewat ya!

1. Buat Folder Project

Kita buat rumah untuk konfigurasi Nextcloud kita:

mkdir ~/nextcloud-server && cd ~/nextcloud-server

2. Buat File docker-compose.yml

Gunakan editor favoritmu (misal `nano`) untuk membuat file konfigurasi:

nano docker-compose.yml

Lalu copy-paste kode "sakti" di bawah ini:

services:
  db:
    image: mariadb:10.6
    restart: always
    command: --transaction-isolation=READ-COMMITTED --binlog-format=ROW
    volumes:
      - ./db:/var/lib/mysql
    environment:
      - MYSQL_ROOT_PASSWORD=ganti_password_root_kamu
      - MYSQL_PASSWORD=ganti_password_user_kamu
      - MYSQL_DATABASE=nextcloud
      - MYSQL_USER=nextcloud

  app:
    image: nextcloud
    restart: always
    ports:
      - 8080:80
    links:
      - db
    volumes:
      - ./data:/var/www/html
    environment:
      - MYSQL_PASSWORD=ganti_password_user_kamu
      - MYSQL_DATABASE=nextcloud
      - MYSQL_USER=nextcloud
      - MYSQL_HOST=db

*Jangan lupa ganti bagian "ganti_password" dengan password rahasiamu sendiri!

3. Jalankan Kontainer

Ini adalah momen kebenaran. Jalankan perintah ini:

docker compose up -d

Sistem akan mendownload semua yang dibutuhkan. Tunggu sampai selesai sambil ngopi-ngopi santai.

4. Akses Lewat Browser

Buka browser kamu dan ketik alamat IP server kamu diikuti port 8080:

http://alamat-ip-servermu:8080

Kamu bakal disambut oleh halaman setup Nextcloud. Buat akun admin, dan voila! Cloud pribadi kamu sudah jadi dan siap diisi foto-foto kenangan atau file kerjaan.

Penting: Folder data dan db di direktori tadi adalah tempat semua datamu disimpan. Jangan sampai terhapus ya, atau datamu bakal hilang selamanya!
