Pendahuluan

ImageMagick memungkinkan kamu mengubah ukuran banyak foto sekaligus tanpa merusak file asli.

Catatan: Hindari mogrify tanpa -path karena akan menimpa file.

Resize Cepat

mkdir -p ../resized
magick mogrify -path ../resized -resize 800x -strip -quality 85 *.jpg

Loop Manual

mkdir -p ../resized
for f in *.jpg; do
  magick "$f" -resize 800x "../resized/$f"
done

Rekursif Subfolder

find . -type f -iname "*.jpg" -print0 |
while IFS= read -r -d '' f; do
  mkdir -p "../resized/$(dirname "$f")"
  magick "$f" -resize 800x "../resized/$f"
done
