Debian Sekarang Sudah “Just Works” Banget!

Ringkasan lengkap diskusi Reddit r/debian (Desember 2025) tentang pengalaman post-installation Debian di Thinkpad X230 dan laptop lainnya.

Pernah nggak sih kamu kangen jaman dulu install Debian harus ngedit sources.list berjam-jam, nge-hunt driver WiFi, dan berdoa supaya Xorg jalan? Nah, kabar baiknya: tahun 2025 itu udah berlalu banget

Intinya Apa Sih?

Debian sekarang (Debian 12/13 “Bookworm/Trixie”) langsung bisa dipakai sehari-hari setelah instalasi. Nggak ada lagi drama “black screen”, “no WiFi”, atau “touchpad mati”. Khususnya buat hardware yang umurnya di bawah 10–12 tahun — termasuk si legenda Thinkpad X230.

Apa yang Berubah dari 10–15 Tahun Lalu?

  • Firmware non-free (yang legal) sudah disertakan di ISO resmi sejak Debian 12 → WiFi Intel, Broadcom, Realtek, Bluetooth langsung nyala.
  • Installer Calamares atau debian-installer sudah jauh lebih pintar mendeteksi hardware.
  • Pilih desktop environment (GNOME, KDE, XFCE, dll) saat instal → reboot → langsung dapat desktop yang cantik dan fungsional.
  • Kernel bawaan sudah cukup baru untuk hardware lama, dan kalau butuh lebih baru tinggal tambah backports.

Pengalaman Orang-orang di Thread Reddit

  • BigRedS: “Install desktop → langsung jadi. Aku cuma install vim, git, sama GNOME extension favorit. X220 dulu nggak ada masalah, X230 pasti sama.”
  • TheRob2D: “Sekarang pengalaman awalnya luar biasa. Aku pakai KDE + kernel backport + driver Nvidia terbaru. Di Thinkpad tua mah nggak perlu apa-apa.”
  • joe_attaboy: Baru install di Beelink mini-PC dan MacBook Air 2015 → “cuma WiFi di MacBook yang butuh satu paket tambahan, sisanya mulus banget”.
  • exarobibliologist: “Dibanding 10–15 tahun lalu, installer jauh lebih user-friendly. Hardware 99% langsung jalan.”

Konfigurasi yang Masih Biasa Dilakuin Orang

  • Tambah repo contrib non-free non-free-firmware
  • Install paket favorit: vim, git, curl, htop, synaptic, chrome/firefox-esr, dll.
  • Nyalain firewall (ufw), printing (cups), atau snapshot (snapper).
  • Yang suka bleeding-edge tambah backports.
  • Setup dotfiles, SSH keys, GPG, shell favorit (zsh, fish, dll).

Buat Thinkpad X230 khususnya?
Semua yang punya X220/X230 bilang: TrackPoint, keyboard legendaris, WiFi Intel, suspend/resume, brightness → semuanya langsung jalan tanpa sentuh apa-apa

Kesimpulan Akhir

Kalau kamu pengen Debian di Thinkpad X230 (atau laptop lain yang nggak terlalu baru):

  1. Download ISO resmi dengan non-free firmware
  2. Pilih desktop environment saat instal
  3. Reboot → langsung bisa kerja/sekolah/ngoding/nonton YouTube
  4. Tinggal tambah paket-paket kesukaanmu, selesai!

Debian 2025 sudah bukan lagi distro “buat yang sabar dan suka ngoprek berjam-jam”. Sekarang dia stabil, minimal bloat, dan tetap just works seperti yang kita harapkan dari distro legendaris ini

Selamat balik ke Debian, bro!
