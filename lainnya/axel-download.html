Axel: Download Accelerator Ringan yang Bikin Ngebut!

Sebuah "curhatan" singkat soal koneksi internet di negara ini, dan penemuan sebuah tool kecil yang ternyata jadi juru selamat.

Saya sudah terlalu jenuh dengan janji-janji dari operator mobile-broadband yang ada di negara ini... 🐒 Meskipun sudah sedikit terobati dengan hadirnya koneksi ADSL dari PT.Telkom. 🤓

Namun, jika saya sedang berada diluar rumah dan ingin mendownload sesuatu, tentunya harus menggunakan koneksi dari mobile-broadband... 😞

Masalah Klasik: Download Pakai WGET

Untuk kegiatan sedot-menyedot biasanya menggunakan wget, namun ternyata ketika diterapkan pada koneksi broadband, harus membuat saya lebih bersabar... sembari menunggu proses sedot rampung, saya iseng buka Synaptic untuk mencari tool download yang mungkin bisa mengatasi kendala lambatnya koneksi mobile-broadband…

Penemuan Ajaib: axel

Akhirnya menemukan sebuah paket bernama AXEL yang sangat ringan dan ampuh... dari keterangan yang didapat ketika mengetikkan axel -h ternyata paket ini menggunakan multiple akses ke sebuah sumber tujuan URL tempat berkas yang akan saya ambil berada…

Intinya, si axel ini "ngeroyok" file yang mau di-download. Dia buka beberapa koneksi sekaligus ke server, terus masing-masing koneksi download bagian file yang beda-beda. Setelah semua bagian selesai, baru deh disatuin lagi. Keren, kan?

Dengan menggunakan tool download ini saya bisa tersenyum... karena bisa mendapatkan kecepatan akses yang lebih baik daripada wget... 😂

Cara Install dan Pakai Axel

Buat kamu yang pengen coba juga, cara install-nya gampang banget kok (khususnya di distro turunan Debian/Ubuntu):

sudo apt update
sudo apt install axel

Cara pakainya juga simpel. Kalau biasanya pakai wget [URL], sekarang ganti jadi axel [URL]. Biar lebih mantap, kita bisa tentuin mau pakai berapa koneksi.

# Contoh download file ISO dengan 10 koneksi sekaligus
axel -n 10 -a -o ubuntu.iso http://example.com/ubuntu-latest.iso
  • -n 10: Ini artinya kita minta axel buka 10 koneksi. Angka ini bisa kamu ubah-ubah, tapi jangan kegedean juga ya, kasihan servernya.
  • -a: Ini cuma buat bikin tampilan progress bar-nya jadi lebih cakep dan informatif.
  • -o ubuntu.iso: Ini buat nentuin nama file output-nya (sama kayak -O di wget).

Selamat mencoba, dan semoga tidak perlu jenuh lagi nungguin download!
