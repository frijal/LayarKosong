What (Apa ini?)

Ini adalah **Editor Artikel JSON Pro**. Singkatnya, ini adalah antarmuka visual (GUI) untuk mengedit file database mentah (`artikel.json`) yang biasanya ribet kalau diedit manual pakai Notepad atau VS Code. Daripada pusing liat kurung siku `[]` dan tanda kutip `""`, mending pakai ini.

Why (Kenapa dibuat?)

Karena manusia tempatnya salah dan lupa. Mengedit JSON manual itu rawan *syntax error* (koma kurang satu aja, satu web bisa error!). Tools ini menjamin struktur data (array di dalam array) tetap **konsisten** dan aman. Plus, jauh lebih enak dilihat mata, kan?

Who (Buat siapa?)

Khusus buat kamu, admin atau pengelola konten yang pakai sistem *static site generator* berbasis JSON (seperti sistem Layar Kosong). Kalau kamu butuh update konten cepat tanpa mau ribet coding, ini jodohmu.

When (Kapan pakainya?)

Setiap kali kamu mau: 1) Nambah artikel baru, 2) Revisi typo di judul/deskripsi, 3) Ganti gambar thumbnail, atau 4) Hapus artikel jadul yang udah nggak relevan. Buka tools ini, upload, edit, download, beres!

Where (Di mana datanya?)

Semua proses terjadi **di browser kamu (lokal)**. Tidak ada data yang dikirim ke server orang lain. Privasi aman 100%. File JSON yang kamu download tinggal kamu upload balik ke repository GitHub atau server hostingmu.

How (Gimana cara pakainya?)

  1. Klik tombol **Buka File JSON** dan pilih file `artikel.json` dari komputermu.
  2. Data akan muncul berdasarkan Kategori.
  3. Klik **Tambah Artikel** di kategori yang sesuai, atau klik tombol **Edit** (ikon pensil) di artikel yang mau diubah.
  4. Kalau mau bikin topik baru, klik **Tambah Kategori** di toolbar atas.
  5. Udah selesai? Klik **Simpan JSON**. File baru akan terunduh otomatis.
