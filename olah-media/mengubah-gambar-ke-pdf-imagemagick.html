Mengubah Banyak Gambar Menjadi Satu PDF dengan ImageMagick

Jika Anda memiliki banyak file gambar terpisah‚Äîseperti hasil scan dokumen atau halaman komik‚Äîakan merepotkan untuk membukanya satu per satu. Solusi praktisnya adalah menggabungkan semua gambar tersebut menjadi satu file PDF.

Langkah 1: Instal ImageMagick

Pastikan Anda sudah memasang paket **ImageMagick** di sistem Anda. Jika belum, instal dengan perintah berikut sesuai distro Anda:

# Untuk Debian/Ubuntu
sudo apt update && sudo apt install imagemagick
# Untuk Fedora
sudo dnf install ImageMagick
# Untuk Arch Linux
sudo pacman -S imagemagick

Langkah 2: Gabungkan Gambar ke PDF

Buka terminal di dalam folder yang berisi gambar-gambar Anda, lalu jalankan perintah `convert`.

convert *.jpg nama_file_output.pdf

Perintah di atas akan mengambil semua file dengan ekstensi `.jpg` di folder saat ini dan menggabungkannya secara berurutan menjadi satu file PDF. Anda bisa mengganti `*.jpg` dengan ekstensi lain seperti `*.png` atau menggunakan pola yang lebih spesifik seperti `halaman-*.jpg`.
