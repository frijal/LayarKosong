IPTV (Internet Protocol Television) itu teknologi yang bikin kita bisa nonton TV, film, atau serial lewat internet‚Äîtanpa perlu antena atau parabola. Bedanya sama Netflix/Roku, IPTV biasanya buat **siaran langsung** (live stream), bukan video on-demand.

Dahulu ada FreeTuxTV, tapi sekarang lebih populer pakai **Hypnotix** (bawaan Linux Mint, tapi bisa kok di Ubuntu/distro lain). Cara pakainya gampang, tampilannya juga kekinian.

‚öôÔ∏è Instalasi Hypnotix di Ubuntu

Install Hypnotix paling gampang lewat **PPA** supaya dependensi aman:

  1. `sudo apt install software-properties-common -y`
  2. `sudo add-apt-repository ppa:kelebek333/mint-tools`
  3. `sudo apt update`
  4. `sudo apt install hypnotix -y`

Buka Hypnotix, cek menu ‚ò∞ ‚Üí **About** buat lihat versi.

üìÇ Kategori Konten

Di halaman utama Hypnotix ada tiga kategori: **TV Channels**, **Movies**, dan **Series**. Jumlah channel bisa beda-beda, ada yang 24 jam, ada yang kadang off.

‚ûï Menambahkan Channel Baru

  1. Unduh file **M3U** dari IPTV-org.
  2. Simpan di `~/.hypnotix/`
  3. Di Hypnotix: menu **Providers** ‚Üí **Add a new provider‚Ä¶**
  4. Pilih **Local M3U File** atau **M3U URL**
  5. Isi nama dan path/URL
