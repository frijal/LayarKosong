Merekam Bagian Layar Desktop dengan FFmpeg

Tutorial praktis untuk merekam area tertentu menggunakan perintah terminal

Bagi Anda yang sering membuat tutorial atau demo di komputer, merekam seluruh layar terkadang tidak efisien. Dengan FFmpeg, kita bisa merekam hanya area tertentu.

Alat-alat yang Dibutuhkan

  1. FFmpeg: Pastikan sudah terpasang (misal: sudo apt install ffmpeg).
  2. Screen Ruler: Untuk menentukan koordinat area yang akan direkam.

Perintah Dasar

ffmpeg -f x11grab -s 700x500 -r 25 -i :0.0+1,1 -sameq output.mpg

Penjelasan Parameter:

  • -f x11grab: Format input untuk X11 (Linux).
  • -s 700x500: Ukuran area rekam (lebar x tinggi).
  • -r 25: Frame rate 25 fps.
  • -i :0.0+1,1: Input desktop dan koordinat area.
  • -sameq: Menjaga kualitas video.
  • output.mpg: Nama file hasil rekaman.

Tips Tambahan

Tekan Ctrl + C untuk menghentikan perekaman. Sesuaikan ukuran, posisi, dan kualitas sesuai kebutuhan.
