Solusi LibreOffice Lambat di Jaringan

Pernah mengalami LibreOffice terasa lambat saat membuka dokumen dari folder jaringan? Ini umum di Linux, dan solusinya ternyata sangat sederhana.

Akar Kendala

Kelambatan disebabkan oleh **GIO** (Glib Input/Output) yang kurang efisien saat mengakses file di jaringan. Kita bisa memaksa LibreOffice menggunakan **GVFS-Fuse** yang jauh lebih cepat.

Langkah-langkah

1. Pastikan `gvfs-fuse` Terpasang

sudo apt install gvfs-fuse

2. Nonaktifkan GIO

sudo sed -i 's/X-GIO-NoFuse=true/#X-GIO-NoFuse=true/' /usr/share/applications/libreoffice*

Ringkasan Cepat

  • Masalah: LibreOffice lambat di folder jaringan (SMB/NFS)
  • Penyebab: GIO tidak efisien
  • Solusi: Gunakan `GVFS-Fuse`
