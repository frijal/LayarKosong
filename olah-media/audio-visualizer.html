Script bash fleksibel: pilih mode, warna, dan resolusi

Narasi dan penjelasan yang kamu minta‚Äîversi santai, lengkap, dan siap tempur. Kamu bisa memilih mode spectrum (equalizer) atau waveform (gelombang), terus set resolusi dan warna sesuai kebutuhan konten.

Script: `audio_visualizer.sh`

#!/bin/bash
# Script untuk membuat video visualisasi audio (spectrum/waveform) dari semua file .mp3 di folder
# Output: file video .mp4 dengan nama sama seperti file audio
echo "Pilih mode visualisasi:"
echo "1) Spectrum (equalizer)"
echo "2) Waveform (gelombang)"
read -p "Masukkan pilihan [1/2]: " choice
read -p "Masukkan resolusi (contoh: 1280x720): " resolution
read -p "Masukkan warna (contoh: white, red, green, rainbow): " color
for f in *.mp3; do
    base="${f%.mp3}"
    echo "‚ñ∂ Memproses $f ..."
    if [ "$choice" == "1" ]; then
        ffmpeg -y -i "$f" \
        -filter_complex "[0:a]showspectrum=s=${resolution}:mode=combined:color=${color}:scale=log[v]" \
        -map "[v]" -map 0:a -c:v libx264 -c:a aac -shortest "${base}_spectrum.mp4"
    elif [ "$choice" == "2" ]; then
        ffmpeg -y -i "$f" \
        -filter_complex "[0:a]showwaves:s=${resolution}:mode=line:rate=25:colors=${color}[v]" \
        -map "[v]" -map 0:a -c:v libx264 -c:a aac -shortest "${base}_waveform.mp4"
    else
        echo "‚ùå Pilihan tidak valid."
        exit 1
    fi
done
echo "‚úÖ Semua file selesai diproses!"
