Fedora: Pasang Codec Tambahan Agar Bisa Bernyanyi 🎵

Apa yang perlu dilakukan

Secara bawaan, Fedora hanya mendukung format media bebas paten seperti OGG. Agar bisa memutar MP3, MP4, MKV, dan lainnya, Anda perlu memasang paket codec tambahan dari repositori pihak ketiga seperti RPM Fusion.

Siapa yang perlu ini

Pengguna Fedora yang ingin memutar format media populer di desktop atau laptop mereka. Cocok untuk pengguna yang ingin pengalaman multimedia lengkap tanpa berpindah distro.

Kapan panduan ini berlaku

Panduan ini mencakup langkah untuk Fedora 16 (catatan lawas) dan cara yang lebih ringkas untuk Fedora 19 ke atas. Untuk rilis modern, gunakan bagian Fedora 19 dan yang lebih baru.

Dimana paketnya berasal

Paket codec biasanya tersedia di repositori RPM Fusion. Pastikan mengunduh dari sumber resmi RPM Fusion untuk keamanan dan kompatibilitas.

Mengapa perlu memasang codec

Tanpa codec non-free, beberapa format populer tidak akan diputar. Menambahkan codec membuat Fedora mampu memutar file multimedia umum dan DVD yang terenkripsi.

Bagaimana caranya

Tahapan untuk Fedora 16

Untuk Fedora 16, proses terdiri dari dua langkah utama: menambahkan repositori RPM Fusion lalu menginstal paket multimedia.

1. Instal Repositori RPM Fusion

sudo yum -y localinstall --nogpgcheck http://download1.rpmfusion.org/nonfree/fedora/rpmfusion-nonfree-release-stable.noarch.rpm http://download1.rpmfusion.org/free/fedora/rpmfusion-free-release-stable.noarch.rpm

2. Instal Paket Multimedia

sudo yum -y install gstreamer gstreamer-plugins-good gstreamer-plugins-bad gstreamer-plugins-ugly libdvdread libdvdnav lsdvd libdvdcss ffmpeg ffmpeg-libs gstreamer-ffmpeg libmatroska xvidcore xine-lib xine-lib-extras-freeworld xine-plugin

Untuk Fedora 19 dan yang Lebih Baru

Proses lebih ringkas: setelah menambahkan RPM Fusion, jalankan dua perintah grup paket berikut.

sudo dnf groupupdate multimedia --setop="install_weak_deps=False" --exclude=PackageKit-gstreamer-plugin
sudo dnf groupupdate sound-and-video

Setelah selesai, Fedora Anda akan bisa memutar banyak format media populer. Selalu periksa dokumentasi RPM Fusion jika ada perubahan paket atau nama grup.
