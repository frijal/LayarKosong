Menggabungkan Berkas PDF dengan PDFtk

Seringkali kita perlu menggabungkan beberapa file PDF menjadi satu dokumen tunggal. Dengan bantuan `PDFtk`, proses ini dapat dilakukan dengan cepat dan mudah langsung dari terminal.

Langkah 1: Instalasi PDFtk

Pastikan Anda sudah memasang paket `pdftk` di sistem Anda. Pada distribusi Linux modern seperti Ubuntu 22.04 ke atas, Anda mungkin perlu menginstal versi Java-nya.

# Untuk Debian/Ubuntu
sudo apt update && sudo apt install pdftk-java
# Untuk Fedora
sudo dnf install pdftk-java
# Untuk Arch Linux
sudo pacman -S pdftk

Langkah 2: Siapkan Berkas dan Jalankan Perintah

Kumpulkan semua berkas PDF yang ingin digabungkan dalam satu folder. Kemudian, buka terminal di folder tersebut dan jalankan perintah berikut:

pdftk berkas1.pdf berkas2.pdf berkas3.pdf cat output berkas-gabungan.pdf

**Penjelasan:**

  • `berkas1.pdf berkas2.pdf ...`: Daftar file PDF yang ingin digabungkan, sesuai urutan.
  • `cat`: Perintah untuk menggabungkan (concatenate).
  • `output berkas-gabungan.pdf`: Menentukan nama file hasil gabungan.

Tips: Jika file Anda memiliki pola nama yang urut (misalnya, `Bab-*.pdf`), Anda bisa menggunakan wildcard untuk menyeleksi semua file sekaligus: `pdftk Bab-*.pdf cat output Buku-Lengkap.pdf`.

Keunggulan PDFtk

  • **Cepat dan Ringan:** Jauh lebih cepat dibandingkan aplikasi GUI yang berat.
  • **Otomatisasi:** Sangat mudah diintegrasikan ke dalam skrip shell untuk alur kerja otomatis.
  • **Banyak Fitur:** Selain menggabungkan, PDFtk juga bisa memecah, memutar, dan mengenkripsi file PDF.
