üß† Tips Terminal: Memaksa Gambar Jadi Rasio 16:9

Menggunakan ImageMagick + Bash Script Otomatis

Kadang kita butuh semua gambar punya rasio yang sama ‚Äî misalnya 16:9 untuk video thumbnail, background, atau halaman web. Dengan **ImageMagick**, kita bisa memaksa ukuran gambar agar pas 16:9, baik dengan *crop* (potong) atau *pad* (tambah ruang kosong). Berikut panduan dan skrip otomatisnya.

üß© Versi Skrip Otomatis

#!/usr/bin/env bash
# paksa-16-9.sh ‚Äî Konversi gambar jadi rasio 16:9 (crop/pad)
set -euo pipefail
MODE="${1:-pad}"  # default: 'pad' (bisa diganti 'crop')
WIDTH=1920
HEIGHT=1080
OUTDIR="output-16-9"
mkdir -p "$OUTDIR"
for img in *.webp *.jpeg *.png *.webp 2>/dev/null; do
  [[ -f "$img" ]] || continue
  echo "üé® Memproses: $img ‚Üí $MODE mode"
  if [[ "$MODE" == "crop" ]]; then
    magick "$img" -resize "${WIDTH}x${HEIGHT}^" -gravity center \
      -extent "${WIDTH}x${HEIGHT}" "$OUTDIR/$img"
  elif [[ "$MODE" == "pad" ]]; then
    magick "$img" -resize "${WIDTH}x${HEIGHT}" -gravity center \
      -background black -extent "${WIDTH}x${HEIGHT}" "$OUTDIR/$img"
  else
    echo "‚ùå Mode tidak dikenal: $MODE (gunakan 'pad' atau 'crop')"
    exit 1
  fi
done
echo "‚úÖ Semua gambar selesai dikonversi ke $OUTDIR/"

üß≠ Cara Pakai

  1. Simpan skrip sebagai `paksa-16-9.sh`.
  2. Jalankan izin eksekusi: 
    chmod +x paksa-16-9.sh
  3. Jalankan di folder gambar: 
    ./paksa-16-9.sh pad   # untuk menambah ruang kosong
./paksa-16-9.sh crop  # untuk memotong gambar agar pas

üîç Penjelasan Parameter

  • **-resize** ‚Üí ubah ukuran gambar, menjaga rasio.
  • **^** ‚Üí memenuhi area target (bisa lebih besar).
  • **-gravity center** ‚Üí potong/letakkan dari tengah.
  • **-extent** ‚Üí ubah kanvas ke ukuran tertentu (memotong atau menambah ruang).
  • **-background black** ‚Üí warna ruang tambahan (bisa white/none).
  • **set -euo pipefail** ‚Üí buat skrip aman dari error.
