Mengambil Semua Gambar dari File PDF dengan pdfimages

Terkadang kita ingin menyimpan semua gambar yang ada dalam sebuah dokumen PDF. Dengan bantuan perintah pdfimages, hal ini dapat dilakukan dengan mudah. Perintah ini merupakan bagian dari paket poppler-utils di Linux.

Perintah Dasar

pdfimages -j file.pdf nama-gambarnya

Opsi -j digunakan untuk menjaga kualitas gambar dengan format JPEG. Hasilnya, semua gambar dari file.pdf akan diekstrak menjadi file gambar dengan awalan nama-gambarnya.

Instalasi poppler-utils

sudo apt install poppler-utils
