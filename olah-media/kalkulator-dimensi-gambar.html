Kalkulator Dimensi Gambar

Input ukuran sisi utama, maka sisi lainnya akan menyesuaikan otomatis sesuai rasio standar.

💡 Info: Alat ini bekerja dengan rumus matematika, perbandingan tetap sederhana.

Jika kamu memilih arah orientasi Horizontal, maka angka yang kamu input dianggap sebagai Lebar (Width), dan sistem akan mencari berapa Tinggi (Height) yang ideal.

Sebaliknya, kalau kamu pilih Vertical, maka nilai angka yang kamu input dianggap sebagai Tinggi, dan di dalam tabel akan menampilkan berapa Lebar yang pas, agar proporsi visual gambar tetap stabil tanpa mengalami distorsi bentuk.


Ini sangat berguna buat optimasi Core Web Vitals (khususnya CLS), karena dengan tau dimensi yang pasti, kamu bisa langsung nulis atribut width dan height di tag HTML kamu secara akurat.
