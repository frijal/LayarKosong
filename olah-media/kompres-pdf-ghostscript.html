Langkah cepat

Perintah inti Ghostscript

Ini perintah yang dapat langsung Anda jalankan untuk mengecilkan PDF:

gs -sDEVICE=pdfwrite \
  -dCompatibilityLevel=1.4 \
  -dPDFSETTINGS=/screen \
  -dNOPAUSE -dQUIET -dBATCH \
  -sOutputFile=NamaBerkasRamping.pdf \
  NamaBerkasGendut.pdf

Kualitas

Pilih level kualitas sesuai kebutuhan

  • **/screen:** Ukuran paling kecil, cocok untuk pratinjau cepat & email.
  • **/ebook:** Keseimbangan bagus untuk baca di layar, kualitas menengah.
  • **/printer:** Lebih tajam untuk cetak, ukuran sedang-besar.
  • **/prepress:** Kualitas tinggi untuk pra-cetak; ukuran paling besar.
  • **Tanpa -dPDFSETTINGS:** Anda bisa atur granular (DPI, JPEGQuality) manual.
  • **Tips:** Untuk dokumen teks, /screen atau /ebook biasanya cukup.

Lanjutan

Kompres kustom (DPI & kualitas JPEG)

Untuk kontrol lebih detail, Anda dapat mengatur resolusi dan kualitas kompresi:

gs -sDEVICE=pdfwrite \
  -dCompatibilityLevel=1.4 \
  -dDownsampleColorImages=true -dColorImageResolution=144 \
  -dDownsampleGrayImages=true  -dGrayImageResolution=144  \
  -dDownsampleMonoImages=true  -dMonoImageResolution=200  \
  -dAutoFilterColorImages=false -dColorImageFilter=/DCTEncode \
  -dJPEGQ=60 \
  -dNOPAUSE -dQUIET -dBATCH \
  -sOutputFile=keluar.pdf masuk.pdf
