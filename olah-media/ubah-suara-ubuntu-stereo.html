Ubah Suara Sistem Ubuntu dengan Audio Buatan Sendiri

Bosan dengan suara notifikasi bawaan Ubuntu? Anda bisa menggantinya dengan file audio Anda sendiri. Prosesnya mudah dan hanya memerlukan beberapa langkah di terminal.

Langkah Aman Mengganti Suara Sistem

1. Temukan Folder Tema Suara

File suara sistem disimpan di dalam direktori tema. Lokasi umumnya adalah /usr/share/sounds/. Anda bisa melihat tema yang tersedia dengan perintah:

ls /usr/share/sounds/

File audio spesifik biasanya ada di dalam subfolder stereo, misalnya: /usr/share/sounds/Yaru/stereo.

2. Cadangkan File Asli

Sebelum melakukan perubahan, selalu buat cadangan file suara asli agar mudah dikembalikan jika diperlukan.

sudo mkdir -p ~/backup-sounds/stereo
sudo cp /usr/share/sounds/Yaru/stereo/*.oga ~/backup-sounds/stereo/

3. Siapkan File Audio Pengganti

Gunakan file audio dengan format OGG atau OGA. Jika file Anda masih dalam format lain (misalnya WAV), Anda bisa mengonversinya menggunakan ffmpeg.

ffmpeg -i input.wav -c:a libvorbis bell.ogg

4. Ganti File dan Muat Ulang Sesi

Salin file audio baru Anda ke direktori tema suara dengan nama yang sama seperti file asli (misalnya, bell.oga).

sudo cp bell.ogg /usr/share/sounds/Yaru/stereo/bell.oga

Terakhir, logout dan login kembali, atau restart sound server agar perubahan diterapkan.

systemctl --user restart pipewire pipewire-pulse
