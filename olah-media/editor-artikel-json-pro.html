JSON Checklist Exporter

Panduan Penggunaan Editor

Editor ini dirancang untuk mengelola database artikel berbasis file `.json` secara offline/online. Anda bisa mendapatkan nama file, dari entri artikel menggunakan visualisasi yang mudah.

  • **Import:** Gunakan tombol `Load URL` atau `Buka File Lokal` untuk memuat file .json Anda.
  • **Export:** Setelah selesai mengedit, tekan tombol `Simpan *.txt` untuk mendapatkan daftar urutan yang berisi nama file dari pilihan yang udah di contreng.
  • **Struktur:** Script ini mengelola JSON dengan format **Object of Arrays**, di mana tiap key adalah nama kategori, dan isinya adalah array yang menampung data dari file .json tersebut.
  • **Data Artikel:** Setiap data diwakili oleh array flat dengan urutan: `[Judul, Nama File, Link Gambar, Tanggal Jam, Deskripsi]`.
