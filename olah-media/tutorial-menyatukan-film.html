Menyatukan Pecahan Film dengan Perintah cat

Apabila Anda mengelola arsip video berdurasi panjang yang terbagi menjadi beberapa fragmen berkas, sistem Linux menyediakan opsi integrasi secara praktis dengan sangat mudah menggunakan perintah cat langsung dari terminal.

Bayangkan Anda memiliki beberapa file seperti di bawah ini dalam satu folder:

Langkah Penyatuan

Prosesnya sangat sederhana:

  1. Buka Terminal di dalam folder yang berisi file-file pecahan tersebut.
  2. Gunakan perintah cat, sebutkan semua nama file secara berurutan, dan arahkan outputnya ke satu file baru menggunakan simbol >.

Contoh Perintah

cat film.part1.avi film.part2.avi film.part3.avi > film-lengkap.avi

Tips: Jika file Anda memiliki pola nama yang urut (misalnya, file.001, file.002), Anda bisa menggunakan wildcard: cat file.* > film-lengkap.mkv.

Hasilnya adalah satu file film utuh pada folder yang sama, tanpa mengubah kualitas video aslinya.
