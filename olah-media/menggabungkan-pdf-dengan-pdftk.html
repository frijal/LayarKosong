Menggabungkan Berkas PDF dengan PDFtk

Panduan Cepat, Efisien, dan Lengkap

Kenapa Perlu Menggabungkan PDF?

Sering kali kita memiliki banyak berkas PDF terpisah: laporan, bab buku, atau dokumen pendukung. Menggabungkannya menjadi satu berkas lebih praktis untuk arsip maupun berbagi.

1. Instalasi PDFtk

Pastikan PDFtk sudah terpasang. Di Linux (Ubuntu/Debian):

sudo apt-get install pdftk

2. Menyiapkan Berkas

  • Kumpulkan semua berkas di satu folder.
  • Gunakan nama sederhana tanpa spasi, misalnya: `laporan-januari.pdf`.

3. Perintah Menggabungkan PDF

pdftk berkas1.pdf berkas2.pdf berkas3.pdf output gabungan.pdf

Contoh: `pdftk Bab1.pdf Bab2.pdf Bab3.pdf output Buku-Lengkap.pdf`
